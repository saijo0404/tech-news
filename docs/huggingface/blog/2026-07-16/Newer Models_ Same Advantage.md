# Newer Models, Same Advantage

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-16
- **原文連結**: https://huggingface.co/blog/Dharma-AI/newer-models-same-advantages

## 核心主題
DharmaOCR 在巴西葡萄牙語光學字符識別任務中，透過專注於特定領域的訓練，即使面對更新的 OCR 模型仍保持顯著優勢。

## 關鍵重點
- **專注訓練優勢**：DharmaOCR 專注於巴西葡萄牙語領域，每個參數都集中於該特定任務，而非分散在多個語言上，這種專注策略使其在特定領域表現更優
- **性能表現**：在葡萄牙語專注評估中，DharmaOCR 得分 0.925，Mistral OCR4 為 0.798，Unlimited-OCR 為 0.7587，差距顯著
- **穩定性改進**：透過 Direct Preference Optimization (DPO) 階段，模型在視覺複雜度下更穩定，減少退化和錯誤，提升生產環境可靠性

## 結論
專注於特定領域的訓練策略比追求通用性更具優勢，即使面對更新的模型架構，這種專注策略仍能保持優勢。DharmaOCR 的未來發展將繼續應用最新技術，但始終聚焦於巴西葡萄牙語 OCR 這一特定領域。
