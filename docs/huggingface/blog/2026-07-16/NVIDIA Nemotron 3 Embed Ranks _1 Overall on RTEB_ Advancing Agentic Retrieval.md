# NVIDIA Nemotron 3 Embed Ranks #1 Overall on RTEB

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-16
- **原文連結**: https://huggingface.co/blog/nvidia/nemotron-3-embed-wins-rteb

## 核心主題
NVIDIA Nemotron 3 Embed 系列模型在 RTEB 排行榜上排名第一，是專為提升代理式檢索（Agentic Retrieval）品質而設計的嵌入模型集合，提供從 1B 到 8B 多種規格選擇。

## 關鍵重點
- **8B 旗艦模型**：在 RTEB 上得 78.5%，MMTEB Retrieval 上得 75.5%，適合高準確性企業 RAG 應用
- **1B 高效模型**：RTEB 得 72.4%，相比前代錯誤率降低 27-28%，降低延遲與成本
- **Blackwell 優化版**：Nemotron-3-Embed-1B-NVFP4 透過 NVFP4 加速提升吞吐量
- **32k 上下文窗口**：支援長文檔與多輪歷史，具備多語言與程式碼檢索能力
- **企業夥伴評估**：Palantir、ServiceNow、Zoom 等多家企業已評估採用，讚賞其在企業代理應用中的準確性提升
- **部署解決方案**：提供 Hugging Face、NVIDIA NIM 微服務、AI Cloud 平台及 vLLM 等企業級部署選項

## 結論
NVIDIA Nemotron 3 Embed 模型展現出卓越的檢索性能與企業部署潛力，特別適合需要高準確性代理式檢索的企業應用。其開放權重與微調配方讓企業可自訂部署，並透過多家企業夥伴的驗證證明其市場價值。
