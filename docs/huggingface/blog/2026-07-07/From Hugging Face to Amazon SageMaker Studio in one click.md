# From Hugging Face to Amazon SageMaker Studio in one click

- **來源**: Hugging Face
- **發布日期**: 2026-07-07
- **原文連結**: https://huggingface.co/blog/amazon/one-click-to-sagemaker-studio

## 核心主題
Hugging Face 與 Amazon SageMaker AI 推出深度整合，開發者可以單次點擊從模型發現直接進入 SageMaker Studio 進行微調或部署。

## 關鍵重點
- 單次點擊整合：在 Hugging Face 模型頁面直接選擇「Customize on SageMaker AI」或「Deploy on SageMaker AI」，即可直接進入對應的 SageMaker Studio 工作流，模型預載入且環境已配置好。
- 預配置權限：新創建的 Studio 環境自動配置完整的 SageMaker AI 權限，包括模型微調、訓練任務、筆記本實驗和端點部署，無需手動設置 IAM 權限。
- GPU 配額可見性：在選擇實例類型時，可直接查看帳戶下可用的 GPU 配額，並可立即看到哪些 G5/G6 實例類型可用。

## 結論
這項整合大幅簡化了從模型發現到企業部署的路徑，開發者可以保持專注於實驗，無需在中斷流程。
