# Hugging Face Models on Foundry Managed Compute

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-07
- **原文連結**: https://huggingface.co/blog/microsoft/foundry-managed-compute

## 核心主題
Microsoft 推出 Foundry Managed Compute，提供 Hugging Face 生態系中經過篩選的開源模型，可單擊部署至 Foundry 平台。

## 關鍵重點
- Microsoft Foundry 平台提供單一端點和 SDK，可部署來自多個來源的模型，包括 Hugging Face 的開源模型
- Foundry Managed Compute 是第三種部署選項，提供預先存儲在 Azure 中的模型權重和經過掃描的運行時
- Hugging Face Models on Foundry 提供每週更新的精選模型目錄，包含文本、視覺、音頻和多模態模型
- 所有模型都經過企業級安全審查，使用 SafeTensors 格式，並由 Microsoft 管理運行時
- 支持全球部署和數據區域部署，提供相同的可觀測性、治理和計費

## 結論
Microsoft 通過 Foundry 平台為企業提供操作開源模型所需的運營層，解決了 Hugging Face 本身不是企業服務平台的問題。
