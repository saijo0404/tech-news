# Introducing Cosmos 3 Edge

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-20
- **原文連結**: https://huggingface.co/blog/nvidia/cosmos3edge

## 核心主題
NVIDIA 推出 Cosmos 3 Edge，一個 40 億參數的開源世界模型，專為邊緣設備設計，幫助機器人理解環境並生成動作。

## 關鍵重點
- 40 億參數開源模型，可在 NVIDIA 邊緣設備（包括 RTX PRO GPU、Jetson 系列）上運行，提供數據中心級性能
- 支持 Jetson T2000/T3000 等新硬件，在 Jetson Thor 上實現 640×360 分辨率下的實時推理，每秒 32 次動作生成
- 在 VANTAGE-Bench 上排名第一，具備最佳視覺分析能力和機器人政策學習表現
- 提供世界建模、政策模式和動作生成能力，可預測動作效果並推斷動作
- 開放源碼，支持後訓練和優化，可針對特定應用進行定制

## 結論
Cosmos 3 Edge 是物理 AI 的開源基礎模型平台，通過共享世界表示連接理解、預測、模擬和動作。開發者可利用開源框架進行後訓練，為特定應用優化模型性能，並部署在邊緣設備上實現實時推理。
