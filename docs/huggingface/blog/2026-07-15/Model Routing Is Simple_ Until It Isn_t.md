# Model Routing Is Simple. Until It Isn't.

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-15
- **原文連結**: https://huggingface.co/blog/ibm-research/model-routing-is-simple-until-it-isnt

## 核心主題
這篇文章探討了 AI 代理系統中模型路由（model routing）的挑戰，指出雖然看似簡單，但實際上涉及複雜的系統優化問題，而非單純的模型分類選擇。

## 關鍵重點
- **成本不僅僅是模型定價**：實際成本取決於模型、工作負載和服務基礎設施的互動。例如，Sonnet 雖然單價較高，但因快取機制受益更多，總成本反而低於 GPT-4.1。
- **複雜性不僅僅是任務難度**：任務難度在路由時往往不可見，且需要同時平衡成本、延遲、專業化、合規性和可靠性等多重因素。
- **延遲不僅僅是模型速度**：用戶體驗的延遲取決於路由本身、基礎設施因素（硬體、快取、負載）和路由粒度，單純比較模型速度無法準確評估。

## 結論
路由不是關於選擇最佳模型，而是關於優化整個系統的運作點。成功的路由系統應將模型選擇視為系統優化問題，同時考慮成本、質量、延遲、合規性和可靠性等多重變數，而非僅依賴分類邏輯。
