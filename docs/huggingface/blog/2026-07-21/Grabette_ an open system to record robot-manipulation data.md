# Grabette: an open system to record robot-manipulation data

- **來源**: Hugging Face Blog
- **發布日期**: 2026-07-21
- **原文連結**: https://huggingface.co/blog/grabette

## 核心主題
Grabette 是一個開源、低成本的手持式夾爪系統，讓任何人都能輕鬆記錄機器人操作演示並自動轉換為機器人可用的數據集，從而推動開放式機器人學習數據集的建設。

## 關鍵重點
- **硬體設計**：Grabette 配備兩顆相機（魚眼鏡頭提供情境資訊、RGBD 相機負責 6-DoF 追蹤），成本約 490€，使用標準 Raspberry Pi 和可購買的感測器組裝。
- **操作簡便**：使用者只需按下按鈕即可記錄操作，無需機器人、實驗室或遠端操作設備，即可獲得機器人可學習的數據。
- **自動化處理**：數據可在瀏覽器中直接處理，自動執行 SLAM 追蹤、驗證軌跡並轉換為 LeRobot 格式數據集。
- **開放生態**：完全開源，數據可上傳至 Hugging Face Hub，與 LeRobot 訓練框架和 OpenArm 評估工具兼容。
- **社區貢獻**：鼓勵全球貢獻者記錄並分享操作演示，共同建立大型、多樣化的開放式機器人操作數據集。

## 結論
Grabette 降低了機器人數據收集的門檻，讓任何人都能參與機器人學習數據集的建設。透過開放式硬體、軟體和數據格式，它將推動機器人學習領域的協同發展，最終解決數據瓶頸問題。
