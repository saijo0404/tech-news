# ScarfBench: Benchmarking AI Agents for Enterprise Java Framework Migration

- **來源**: Hugging Face Blog
- **發布日期**: 2026-06-30
- **原文連結**: https://huggingface.co/blog/ibm-research/scarfbench

## 核心主題
ScarfBench 是一個評估 AI 代理在企業 Java 框架遷移任務中表現的開源基準測試，專注於 Spring、Jakarta EE 和 Quarkus 之間的遷移。

## 關鍵重點
- 框架遷移比傳統代碼生成更具挑戰性，需要保持行為、適應構建系統並處理運行時依賴關係
- 即使是最先進的代碼代理在框架遷移任務中的行為成功率也低於 10%
- 遷移失敗通常源於環境和工具問題，而非代碼轉換本身

## 結論
框架現代化的最大挑戰不是翻譯 Java 代碼，而是管理跨配置、基礎設施和運行時環境的依賴關係網絡。
