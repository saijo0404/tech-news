# Investing in multi-agent AI safety research

- **來源**: DeepMind Blog
- **發布日期**: 2026-06-11
- **原文連結**: https://deepmind.google/blog/investing-in-multi-agent-ai-safety-research/

## 核心主題
Google DeepMind 與 Schmidt Sciences、Cooperative AI Foundation、ARIA 及 Google.org 聯合宣布推出高達 1,000 萬美元的研究資助計畫，全球徵求提案，聚焦於多 Agent AI 系統的集體安全與風險治理研究。

## 關鍵重點
- **多 Agent 世界的集體安全挑戰**：隨著 AI 技術擴展，數百萬由不同組織建構的 AI Agent 將在各個數位環境中互動、溝通、談判與交易，產生難以預期的「湧現」（emergent）行為——如不可預測的經濟活動潮湧或新的安全挑戰，而現有安全評估多孤立分析單一模型，缺乏預測、測量與監控這些集體行為轉化的工具。
- **四大優先研究領域**：（1）沙盒與測試床：建置真實且可重現的模擬環境（如虛擬市場、模擬生態系、跨組織工作流）來評估與比較多 Agent 安全技術；（2）Agent 網絡科學：研究 Agent 群體的集體能力如何湧現與擴展、網絡如何失效或變得波動，以及檢測危險的群體層級特性；（3）強化 Agent 基礎設施：壓力測試身份、聲譽與承諾協議以確保跨平台 Agent 互動安全；（4）監督與控制：開發監控已部署 Agent 群體並降低集體危害的方法。
- **跨機構合作與申請時程**：此計畫結合 Schmidt Sciences 的「可信賴 AI 科學」與「AI Agent」計畫、ARIA 的「規模化信任」計畫，以及 Google.org 的支持；申請截止日期為 2026 年 8 月 8 日，獲選名單預計於 2026 年秋季公布。

## 結論
多 Agent 安全研究已進入關鍵轉折點，Agent 互動的複雜性正超過現有安全模型的承受能力；DeepMind 呼籲全球學術與獨立研究者參與，共同建構多 Agent 未來的安全基礎，強調單一實驗室無法獨力解決此挑戰。
