# Start building with Nano Banana 2 Lite and Gemini Omni Flash

- **來源**: Google DeepMind
- **發布日期**: 2026-06-30
- **原文連結**: https://deepmind.google/blog/start-building-with-nano-banana-2-lite-and-gemini-omni-flash/

## 核心主題
Google DeepMind 推出兩款新模型 Nano Banana 2 Lite（快速圖像生成）和 Gemini Omni Flash（高品質視頻生成與編輯），幫助開發者更快構建多媒體應用。

## 關鍵重點
- **Nano Banana 2 Lite**：家族中最快速、成本效益最高的圖像模型，4 秒內完成圖像生成，適合快速原型開發和批量處理，成本為每張 1K 圖像 $0.034
- **Gemini Omni Flash**：支持對話式視頻編輯，可結合文本、圖像和視頻輸入進行高品質視頻生成，價格為每秒 $0.10，與 Veo 3.1 Fast 相同
- **開發者平台**：兩款模型均可在 Google AI Studio、Gemini API 和 Gemini Enterprise Agent Platform 上使用，並已推出至消費者端產品（Gemini app、Google Flow 等）

## 結論
開發者可以將這兩款模型串聯起來，先使用 Nano Banana 2 Lite 生成圖像，再使用 Gemini Omni Flash 將其動畫化為視頻，實現完整的端到端多媒體體驗。
