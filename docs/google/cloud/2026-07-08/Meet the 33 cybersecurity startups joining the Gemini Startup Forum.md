# Meet the 33 cybersecurity startups joining the Gemini Startup Forum

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-09
- **原文連結**: https://cloud.google.com/blog/products/identity-security/meet-the-33-cybersecurity-startups-joining-the-gemini-startup-forum/

## 核心主題
Google 宣布其 Gemini Startup Forum 選定首批 33 家 cybersecurity 創業公司，這些公司專注於 AI 整合的安全解決方案，涵蓋從自主代理保護到後量子密碼學等六個專業領域。

## 關鍵重點
- **AI agent security and governance**：包括 Capsule Security、Evoke Security、Manifold Security 等公司，專注於自主 AI 代理的運行時保護和治理。
- **Application security and vulnerability management**：包括 Aisy、Alt Security、Arcjet 等公司，專注於應用安全漏洞管理和自動化工具。
- **Cloud, network, and infrastructure security**：包括 CloudFence、CyberSeQ、Huskeys 等公司，專注於雲網絡和基礎設施安全。
- **Endpoint security and data protection**：包括 Bold Security、Glow、Jazz 等公司，專注於端點安全和數據保護。
- **SOC automation and offensive security**：包括 COGNNA、Latent Defense、Mate Security 等公司，專注於安全運維自動化和安全測試。
- **Security infrastructure and specialized services**：包括 Netsec、Revelum、Synqly 等公司，專注於安全基礎設施和專業服務。

## 結論
這些創業公司通過 Google 的 Gemini Startup Forum 獲得支持，將幫助構建更安全、更具韌性的數字基礎設施，並推動從周界防禦向自主 AI 代理安全的轉變。
