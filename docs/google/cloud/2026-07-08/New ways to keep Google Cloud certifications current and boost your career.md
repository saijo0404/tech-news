# New ways to keep Google Cloud certifications current and boost your career

- **來源**: Google Cloud Blog Training and Certifications
- **發布日期**: 2026-07-09
- **原文連結**: https://cloud.google.com/blog/topics/training-certifications/new-ways-keep-google-cloud-certifications-current/

## 核心主題
Google Cloud 推出新的認證更新機制，透過技能徽章和基於工作的培訓來簡化認證更新流程，幫助專業人士更靈活地保持認證現行。

## 關鍵重點
- **傳統考試模式轉變**：不再強制每兩年參加多小時的監考考試，改為更靈活的認證更新方式
- **Google Skills 新機制**：允許使用者選擇特定課程和技能徽章來更新認證，可優先選擇與職能最相關的內容
- **技能徽章優勢**：透過互動式實作實驗室驗證實際應用能力，比單純上課更快完成認證更新
- **自動延長機制**：完成認證活動後，認證會自動延長一年，無需重新參加考試
- **適用對象**：適用於 Cloud Digital Leader、Associate Cloud Engineer、Professional Cloud Architect 和 Professional Data Engineer 等認證持有者

## 結論
在技術快速變化的時代，Google Cloud 透過新的認證更新機制，讓專業人士能更靈活地保持技能現行，同時提升職業競爭力。主動更新認證不僅是證明知識，更是向雇主展示持續學習能力的「活認證」。
