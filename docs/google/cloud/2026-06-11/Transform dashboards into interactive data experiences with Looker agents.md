# Transform dashboards into interactive data experiences with Looker agents

- **來源**: Google Cloud Blog (Looker)
- **發布日期**: 2026-06-11
- **原文連結**: https://cloud.google.com/blog/products/business-intelligence/dashboard-agents-in-looker/

## 核心主題
Looker 推出 Dashboard Agents 預覽功能，在儀表板中內嵌基於 Gemini 的對話式資料代理，讓使用者以自然語言直接與儀表板互動，突破傳統儀表板缺乏互動性的限制。

## 關鍵重點
- **互動式代理導向探索**：使用者可在儀表板中點擊 Gemini 圖示直接提問，代理可讀取已套用的篩選器、交叉篩選器與預先精選圖磚，確保回覆高度相關且準確；若查詢需要更多資料，代理還能存取底層 Explores 以挖掘更深層洞察，並搭配圖表與自然語言解釋。
- **高度可配置的企業語意層**：資料分析師可在 Looker 語意層之上，以自然語言指令為代理提供額外上下文，定義代理如何詮釋獨特的商業邏輯並針對目標受眾調整回覆內容，使分析團隊能在維持一致可靠分析環境的前提下，以自服務模式應對日益增長的資料需求。
- **繼承的信任治理與透明度**：代理在生成洞察時會明確展示中間推理過程、引用的儀表板圖磚與套用的篩選器，確保透明度；所有存取皆由 Looker 現有治理模型與標準權限管理，使用者僅能查看其獲授權的資料；未來將支援 iframe 嵌入，讓代理可整合至企業門戶或應用程式中。

## 結論
Looker Dashboard Agents 以 Looker 版本 26.08.11 起提供預覽，管理員可在 Gemini in Looker 設定中開啟「Enable Chat with Dashboard」即啟用；這項功能讓傳統靜態儀表板升級為互動式資料體驗，兼顧自服務探索的靈活性與企業治理的可信度。
