# 以 Looker 代理將儀表板轉型為互動式資料體驗

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-11
- **原文連結**: https://cloud.google.com/blog/products/business-intelligence/dashboard-agents-in-looker/

## 核心主題
Google 推出 Looker 儀表板代理（Dashboard Agents）預覽版，將 Gemini 對話式 AI 直接嵌入儀表板介面，讓用戶能以自然語言提問並即時獲得上下文相關的商業洞察，打破傳統靜態儀表板的限制。

## 關鍵重點
- **互動式代理主導調查**：用戶點擊儀表板上的 Gemini 圖標即可用自然語言提問，代理會根據使用者已套用的篩選器、交叉篩選與預建圖表等上下文，產生高度相關的答案；當單一圖表資訊不足時，代理可進一步存取底層 Explores 挖掘更深層洞察，並搭配圖表與自然語言說明。
- **為企業客製化代理行為**：資料分析師可透過自然語言指令為代理補充業務語意上下文，定義代理如何詮釋獨特的業務邏輯，確保不同目標受眾獲得一致的回應，使分析團隊能在維持品質的前提下擴展自我服務式資料分析的規模。
- **繼承 Looker 治理模型的信任機制**：代理在生成洞察時會明確展示推理過程、參考的儀表板圖表與套用的篩選器，提供透明度；同時繼承 Looker 標準權限模型，確保使用者僅能存取其被授權的資料與洞察，且 Google 正開發 iframe 嵌入功能以利整合至企業門戶。

## 結論
Looker 儀表板代理將傳統靜態儀表板升級為對話式、可追問的互動式資料體驗，讓非技術用戶也能在既有的治理框架內自行探索資料，縮小業務用戶與資料分析團隊之間的橋樑，為企業 BI 開啟新一代自我服務分析的新篇章。
