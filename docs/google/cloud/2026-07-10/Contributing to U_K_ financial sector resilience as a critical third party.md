# 貢獻於英國金融業韌性：作為關鍵第三方

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-11
- **原文連結**: https://cloud.google.com/blog/products/identity-security/contributing-to-uk-financial-sector-resilience-as-a-critical-third-party/

## 核心主題
Google Cloud EMEA 已被英國財政部指定為英國金融業的關鍵第三方（CTP），將直接接受英國金融監管機構監督，以確保英國金融系統的穩定性和信心。

## 關鍵重點
- **監管地位**: Google Cloud EMEA 被指定為英國金融業的關鍵第三方，將由英國銀行、審計監管機構（PRA）和金融行為監管局（FCA）共同監管。
- **合作目標**: Google Cloud 支持建立跨行業的營運韌性，並與英國金融監管機構進行建設性對話，以確保英國金融系統穩定。
- **客戶支援**: Google Cloud 持續協助英國企業滿足營運韌性、外包和第三方風險管理要求，並提供相關白文件和映射文件。

## 結論
Google Cloud EMEA 被指定為關鍵第三方標誌著與英國金融監管機構合作的新一階段，將加強透明度與保障，確保 Google Cloud 成為最安全、可擴展且具韌性的數位轉型平台。
