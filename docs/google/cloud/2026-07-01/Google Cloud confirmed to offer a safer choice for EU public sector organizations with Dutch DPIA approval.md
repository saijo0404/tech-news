# Google Cloud confirmed to offer a safer choice for EU public sector organizations with Dutch DPIA approval

- **來源**: Google Cloud Blog Security & Identity
- **發布日期**: 2026-07-02
- **原文連結**: https://cloud.google.com/blog/products/identity-security/google-cloud-confirmed-to-offer-a-safer-choice-for-eu-public-sector-organizations-with-dutch-dpia-approval/

## 核心主題
Google Cloud 獲得荷蘭政府 DPIA 認證，確認其隱私基礎，使荷蘭公共部門可以安全使用 Google Cloud。

## 關鍵重點
- 荷蘭政府戰略供應商管理機構 SLM Rijk 已完成對 Google Cloud 的嚴格數據保護影響評估 (DPIA)
- DPIA 評估確認實施建議措施後，沒有已知的高數據保護風險
- 荷蘭中央公共部門現在可以正式使用 Google Cloud，並有清晰的隱私評估途徑
- 這也建立在之前荷蘭 DPIA 對 Google Workspace 成功完成的基礎上
- Google Cloud 承諾繼續投資隱私增強技術和客戶控制，幫助客戶滿足合規義務

## 結論
Google Cloud 通過荷蘭政府 DPIA 認證，為歐洲公共部門組織提供了更安全的選擇，並加強了對隱私保護的信任。
