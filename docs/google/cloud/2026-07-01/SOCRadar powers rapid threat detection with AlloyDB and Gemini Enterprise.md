# SOCRadar powers rapid threat detection with AlloyDB and Gemini Enterprise

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-02
- **原文連結**: https://cloud.google.com/blog/products/databases/socradar-powers-rapid-threat-detection-with-alloydb-and-gemini-enterprise/

## 核心主題
SOCRadar 透過遷移至 Google Cloud AlloyDB 和整合 Gemini Enterprise，實現了更快的威脅檢測能力和大幅降低的營運成本。

## 關鍵重點
- **性能大幅提升**: 遷移至 AlloyDB 後，查詢速度提升 20 倍，資料處理速度提升 3.2 倍，成功處理「三重威脅」工作負載
- **AI 驅動的警報過濾**: 整合 Gemini Enterprise Agent Platform 自動過濾虛假警報，解決 SOC 警報疲勞問題
- **營運成本大幅降低**: 自動儲存管理釋放 45TB 儲存空間，DBA 工作量減少 75%，實現自動化安全運維

## 結論
透過結合高性能資料庫與 AI 能力，SOCRadar 成功建立了自動化、未來導向的全球網路安全防禦架構，為客戶提供即時威脅洞察。
