# What's New with Google Cloud

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-01
- **原文連結**: https://cloud.google.com/blog/topics/inside-google-cloud/whats-new-google-cloud/

## 核心主題
Google Cloud 在 AI 平台、API 管理、基礎設施和開發者工具方面推出多项重大更新，強化 AI 代理治理能力與雲端服務能力。

## 關鍵重點
- **AI 模型更新**：Claude Sonnet 5 推出取代 Sonnet 4.6，Gemini 3.1 Pro 與 Flash-Lite 正式推出，提供增強推理能力與高成本效益方案
- **Apigee AI 解決方案**：推出 MCP 統一治理、自動化 API 閘道配置與 AI Portals，支援 YAML 藍圖管理與自主代理管理
- **基礎設施優化**：Run:ai Model Streamer 原生支援 TPU 加載速度提升 2 倍，分數 G4 VM 一般可用支援 GPU 切片
- **開發者工具擴展**：Workbench Notebooks 擴展支援 VS Code，Cloud Location Finder 支援多雲規劃，BigQuery Graph 支援供應鏈數字孿生建模
- **資料與儲存**：Datastream 與 Knowledge Catalog 整合提供集中式資料管理，Firestore 推出進階查詢引擎支援 100 個新查詢功能
- **安全與治理**：強化 AI 代理治理架構包含審計日誌與工具存取控制，支援混合環境 Kubernetes 秘密存取
- **社群活動**：多場技術研討會涵蓋 AI 治理、安全與部署，Gemini Live Agent Challenge 競賽獎金超過 8 萬美元

## 結論
Google Cloud 持續強化 AI 平台能力與 API 管理解決方案，同時優化基礎設施與開發者體驗，為企業提供全面的雲端 AI 服務生態系統。
