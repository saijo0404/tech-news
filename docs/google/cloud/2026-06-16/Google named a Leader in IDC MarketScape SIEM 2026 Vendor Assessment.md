# Google named a Leader in IDC MarketScape SIEM 2026 Vendor Assessment

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-16
- **原文連結**: https://cloud.google.com/blog/products/identity-security/google-named-a-leader-in-idc-marketscape-siem-2026-vendor-assessment/

## 核心主題
Google 獲 IDC 評選為 2026 年全球 SIEM 廠商評估中的「領先者」（Leader），肯定 Google Security Operations 整合 Mandiant 專家經驗、Gemini AI 自動化作戰與全域威脅情資的綜合實力。

## 關鍵重點
- **Gemini AI 代理 agent 驅動安全作業**：Alert Triage and Investigation agent 自動蒐集證據、執行關聯搜尋並產出透明判決，降低分析師負擔；Google Cloud Next 宣布的額外 agent 更將自動化工作流程從事件分級延伸至主動威脅搜尋與規則生成，讓分析師以自然語言進行複雜搜尋，秒級生成檢測規則與自動化回應腳本。
- **垂直整合 AI 架構與 Mandiant 檢測內容**：Google 自製矽晶片、營運基礎設施、透過 DeepMind 開發 Gemini 基礎模型，並將內部安全專家知識編碼進 agent 評估循環，避免依賴第三方模型 API，取得更嚴密的迭代週期；Mandiant 分析師撰寫的高階檢測規則集已映射至 MITRE ATT&CK 框架，客戶回饋其开箱即用、偵測準確率高。
- **大規模資料搜尋與全球威脅情資整合**：Google Security Operations 的統一資料湖結合全時態 UDM 搜尋與多階段交叉連結查詢，分析師可無效能衰退地查詢完整保留週期資料；整合 Mandiant 前線經驗、VirusTotal 社群全球觸及與 Google 自身服務可見度，透過 Mandiant Hunt 主動協助防禦者搜尋未檢測攻擊與敵手戰術。

## 結論
IDC 的領先者評級不僅是對 Google SIEM 技術能力的肯定，更凸顯在 AI 對手以前所未有的速度與規模發動攻擊的時代，唯有將底層硬體、基礎模型、威脅情資與安全作業深度整合的平台，才能真正讓防禦者以更少的警報做出更精準的決策。
