# What Google Cloud announced in AI this month

- **來源**: Google Cloud Blog
- **發布日期**: 2025-06-30
- **原文連結**: https://cloud.google.com/blog/products/ai-machine-learning/what-google-cloud-announced-in-ai-this-month/

## 核心主題
Google Cloud 在 AI 領域的月度重點更新，涵蓋平台升級、安全防護、開發工具、數據架構及客戶應用案例。

## 關鍵重點
- **平台升級**：推出 Gemini Enterprise Agent Platform、Gemini 3.5 系列及 Gemini Omni，強化 AI 代理的構建、擴展與優化能力
- **AI 安全防護**：推出 Google AI Threat Defense 全自動 cybersecurity 解決方案，以及 CodeMender AI 安全代理自動修復程式碼漏洞
- **開發工具完善**：Google Antigravity 2.0、Agent Designer 無程式碼設計工具及 Managed Agents API 提供完整的代理開發生態
- **數據與知識架構**：Agentic Data Cloud 支援跨雲湖倉，Open Knowledge Format (OKF) 標準化 LLM 知識格式
- **生態合作**：與 Apple 合作擴展 Private Cloud Compute，支持 Claude Fable 5 等第三方模型及 Adobe、Atlassian 等第三方代理
- **客戶應用案例**：已收集超過 1,300 個客戶 AI 應用案例，包括 Cloud Atelier 虛擬購物體驗和 Gemini Spark 24/7 個人 AI 助手

## 結論
Google Cloud 持續強化 AI 平台能力、安全防護和開發工具，並通過大量客戶應用案例驗證其價值，同時在 2025 年推出 Universal Commerce Protocol 等多項創新功能，推動 AI 商業化與多模態模型擴展趨勢。
