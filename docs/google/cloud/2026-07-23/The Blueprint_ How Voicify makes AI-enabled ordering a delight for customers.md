# The Blueprint: How Voicify makes AI-enabled ordering a delight for customers

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-24
- **原文連結**: https://cloud.google.com/blog/topics/customers/bringing-delight-to-customer-phone-calls-with-ai/

## 核心主題
Voicify 利用 Google Cloud 的 AI 技術（特別是 Gemini Enterprise）為餐廳和醫療機構的電話服務提供精確、快速且安全的 AI 助手，解決高呼叫量下的服務挑戰。

## 關鍵重點
- 使用 Gemini Flash 和 Gemini Enterprise Agent Platform 大幅降低延遲，提升回應速度，減少客戶掛斷率
- 透過 Vertex AI 的預留通量和按使用量付費功能，有效管理流量尖峰並降低成本，同時確保服務穩定性
- 確保 HIPAA、SOC2、ISO27001 和 PCI 合規，滿足醫療機構的安全與合規需求
- 客戶上線時間從兩週縮短至一兩天，大幅提升服務導入效率

## 結論
Google Cloud 的企業級 AI 解決方案幫助 Voicify 成功解決電話服務中的關鍵挑戰，為餐廳和醫療機構帶來更佳的客戶體驗，並為其他企業提供可參考的 AI 實施路徑。
