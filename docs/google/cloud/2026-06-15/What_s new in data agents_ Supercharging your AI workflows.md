# Google 打造新一代資料智能體（Data Agents），加速 AI 工作流

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-15
- **原文連結**: https://cloud.google.com/blog/products/data-analytics/new-data-agents-across-the-agentic-data-cloud/

## 核心主題
Google 發布一系列全新的資料智能體與工具，將 AI 能力全面融入資料處理流程，讓企業能透過自然語言與資料庫互動，並自動化資料工程、資料科學與資料庫管理等日常任務。

## 關鍵重點
- **對話式分析（Conversational Analytics）全面擴充**：BigQuery、Lakehouse、AlloyDB、Spanner、Cloud SQL 及 Looker 均支援以自然語言查詢資料，使用者可直接用日常語言提問，無需撰寫 SQL。Looker Embedded 版本已正式發布，可將對話式 AI 直接嵌入自有應用程式。
- **多款 Google 自建資料智能體登場**：包含自動化資料管線建置的「資料工程智能體」、加速模型開發的「資料科學智能體」、「資料庫可觀測性智能體」與「資料庫上線智能體」等，涵蓋資料工程師、資料科學人與 DBA 的日常需求。此外還有 Deep Research Agent 和 Data Insights Agent，提供跨結構化與非結構化資料的統一洞察。
- **開放標準與開發工具生態系**：推出 Data Agent Kit（IDE/CLI 內的標準化開發工具組）、Managed MCP Servers（讓 AI 模型安全連接資料庫）、MCP Toolbox 1.0 正式版，以及 QueryData 功能，將自然語言轉為資料庫查詢的準確率接近 100%。

## 結論
Google 透過 Agentic Data Cloud 建構了一套從底層晶片到 Gemini 模型的端到端 AI 原生資料架構。這一波推出的資料智能體與開發工具，不僅大幅降低企業使用 AI 分析資料的門檻，也強調治理、安全與開放標準，讓組織能在可控的前提下，將即時資料轉化為可操作的商業智能。
