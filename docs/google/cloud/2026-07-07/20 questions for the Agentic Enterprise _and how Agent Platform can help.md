# 20 Questions for the Agentic Enterprise and how Agent Platform can help

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-07
- **原文連結**: https://cloud.google.com/blog/products/ai-machine-learning/20-questions-for-the-agentic-enterprise/

## 核心主題
Google Cloud 推出 Gemini Enterprise Agent Platform，透過 20 個關鍵問題協助企業建立安全、可擴展的 AI 代理系統，涵蓋建立、擴展與優化三個核心階段。

## 關鍵重點
- **建立階段**：支援四級代理開發工具梯階（Agent Studio → Managed Agents API → Antigravity 2.0 → ADK 2.0），並根據目標使用者選擇合適架構
- **擴展階段**：使用 Model Context Protocol (MCP) 連接企業數據，採用 A2A 協議實現跨框架代理互通，並透過 Agent Runtime 實現彈性自動擴展
- **優化階段**：透過隔離沙箱限制代理風險，結合 Guardrails 確保符合企業規範，並採用分級模型策略優化成本
- **資料存取權限對齊**：透過代理身份、獨立身份或授權授權模式，確保代理僅存取授權資料
- **代理濫用管理**：使用中央代理註冊表追蹤所有活躍代理，避免資料碎片化
- **雙層政策架構**：結合 IAM 政策與語意政策，即時驗證使用者指令合規性
- **Model Armor**：整合提示防護服務，防止資料洩漏與提示注入攻擊
- **Agents CLI**：提供統一命令工具，簡化代理開發、測試與部署流程

## 結論
透過早期解決安全與運營問題，並採用統一開發、管理、治理與優化工具，可建立真正具商業價值的代理系統。
