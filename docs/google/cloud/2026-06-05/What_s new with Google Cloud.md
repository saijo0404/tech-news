# Google Cloud 最新動態：從 AI Agent 治理到雲端基礎建設的全面升級

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-05
- **原文連結**: https://cloud.google.com/blog/topics/inside-google-cloud/whats-new-google-cloud/

## 核心主題
Google Cloud 於 2026 年 1 至 6 月期間密集發布多項重磅更新，涵蓋 Gemini 系列模型迭代（Flash-Lite、3.1 Pro、3.5）、Apigee MCP 全面上市、BigQuery Graph 數位分身、Cloud Run 伺服器無 AI 推論、Nano Banana 2 影像生成等，全面推進企業級 AI Agent 部署與雲端基礎建設優化。

## 關鍵重點
- **模型生態系爆發式擴展與邊緣部署能力**：Gemini 3.1 Flash-Lite 以極高效率與成本支援高容量翻譯與內容審核；Gemini 3.1 Pro 提供更強複雜問題解決能力；Claude Opus 4.8 登陸 Gemini Enterprise Agent Platform 支援代理程式編碼；Anthropic Claude Opus 4.8 加入企業部署選項；Google AI Edge Portal 新增支援 120 款以上 Android 裝置的 LLM 效能基準測試與除錯能力；Fractional G4 VMs 正式上線，以 1/8、1/4、1/2 GPU 彈性切片提供 NVIDIA RTX PRO 6000 Blackwell GPU 的靈活的 AI 與圖形工作負載。
- **Apigee + MCP：企業級 AI Agent 治理核心**：Apigee Model Context Protocol（MCP）正式全面上市，讓開發者能將企業 API 轉為 AI 代理工具，無需額外基礎設施；Apigee 持續推出延伸代理程式閘道模式、細粒度授權（FGA）、安全 token 交換與 Model Armor 防提示注入等功能；API Hub 新增與 API Gateway 整合自動集中元資料，以及 AI 增強 API 文件補件；OpenAPI v3 原生支援 API Gateway 與 Cloud Endpoints；Kubernetes Secrets 支援 Apigee hybrid 安全憑證管理。
- **資料分析、無運算與開發者體驗全面升級**：BigQuery Graph 讓企業能將實體資產轉為節點與邊緣的互動地圖，建立物理世界數位分身以優化供應鏈管理；BigQuery Studio 內建 Gemini 助手升級為情境感知分析夥伴；Firestore 企業版導入全新管線操作引擎（百種新查詢功能、無索引查詢）；Cloud Run Worker Pools 與 CREMA 開源自動縮放器正式上線，支援非 HTTP 背景 AI 推論任務；VS Code 推出 Google Cloud Workbench Notebooks 擴充功能，結合本地 IDE 生產力與雲端可擴展基礎設施。

## 結論
從 2026 年初至今，Google Cloud 的更新脈絡清晰——以「模型多元化 + Agent 治理 + 資料基礎建設」三支柱推進企業 AI 從實驗走向生產級部署，Apigee MCP 的正式上市標誌著 AI 代理程式安全治理進入成熟階段，而 BigQuery Graph、Firestore 新引擎與 Cloud Run Worker Pools 則為 AI 時代的資料與運算需求提供了堅實底盤。
