# Key findings from the 2026 Public Sector M-Trends report and beyond

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-14
- **原文連結**: https://cloud.google.com/blog/topics/public-sector/key-findings-from-the-2026-public-sector-m-trends-report-and-beyond/

## 核心主題
2026 年公共部門面臨的威脅演變為機器速度攻擊，需要從傳統 perimeter 防禦轉向持續驗證的防禦模式。

## 關鍵重點
- **22 秒交接時間**：攻擊者從初始立足點到轉交勒索軟體操作員的時間中位數僅為 22 秒，使傳統人工分級無效。
- **持續性悖論**：國家支持的情報行動者可能滯留超過五年，挑戰標準 90 天監控保留政策。
- **虛擬化堆疊**：攻擊者轉向虛擬化管理平面，使用「快照掛載」技術繞過客戶端安全工具。
- **SaaS 多米諾效應**：第三方雲端工具整合成為威脅向量，非人類身份（如服務帳戶）被利用引發連鎖反應。
- **語音詐騙激增**：語音詐騙（vishing）佔全球感染的 11%，針對政府服務台進行社會工程攻擊。

## 結論
公共部門需要從被動分級轉向主動的持續驗證，利用 Google Cloud 的安全技術（如身份作為新邊界、代理防禦、強化基礎設施）來建立情境感知韌性。
