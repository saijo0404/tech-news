# Accelerating automotive innovation with C4A-metal and Panasonic Automotive vSkipGen

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-21
- **原文連結**: https://cloud.google.com/blog/topics/partners/panasonic-automotive-vskipgen-runs-on-axion-based-c4a-metal/

## 核心主題
這篇文章介紹了 Panasonic Automotive 的 vSkipGen 虛擬化平台如何在 Google Cloud 的 C4A-metal 裸金屬服務器上運行，以加速汽車軟件開發和驗證。

## 關鍵重點
- vSkipGen 是一個下一代 cockpit 虛擬化平台，整合了 Panasonic Automotive 的 Unified HMI 遠端 GPU 離線技術，支持 Android Automotive OS 和 Android SDV。
- C4A-metal 是 Google Cloud 基於 Axion 架構的裸金屬服務器，提供 96 vCPU、兩種 DDR5 記憶體配置（384GB 和 768GB）以及高達 100Gbps 的網路帶寬。
- 通過使用 C4A-metal 和 vSkipGen，汽車製造商可以在雲端構建和驗證完整的 AAOS 堆疊，無需依賴昂貴的物理原型。

## 結論
這種解決方案使汽車製造商能夠在硬件獨立的、雲本環境中構建、測試和驗證完整的 AAOS 堆疊，從物理依賴轉向可擴展的数字孿生，加速下一代駕駛艙平台的上市時間。
