# Shift into high gear with agents: Securing the software-defined vehicle

- **來源**: Google Cloud Blog Security & Identity
- **發布日期**: 2026-07-07
- **原文連結**: https://cloud.google.com/blog/products/identity-security/shift-into-high-gear-with-agents-securing-the-software-defined-vehicle/

## 核心主題
Google Cloud 與 Valtech 合作推出 Nexus SDV 平台，利用 AI 和代理技術重新定義軟體定義車輛的安全與智能。

## 關鍵重點
- Nexus SDV 是一個可擴展的 AI 驅動車輛平台，可管理高達 1 億台設備，與 Android Automotive OS 深度整合
- 使用 Gemini 模型和 Gemini Enterprise Agent Platform 進行即時數據分析，提供自主決策和超個人化駕駛輔助
- 採用 Arm 架構計算和 Bigtable 優化存儲，顯著降低總擁有成本 (TCO)
- 實施深度防禦安全模型，包括 mTLS、零信任架構、身份託管、秘密管理等
- 基於 Google Cloud 安全基礎設施，提供符合合規要求的安全 AI 框架

## 結論
Nexus SDV 代表汽車智能的新時代，提供代理、安全和成本效益的平台，讓製造商能在開源框架中充分利用 AI 能力。
