# Now in preview: Find and fix software vulnerabilities with CodeMender

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-21
- **原文連結**: https://cloud.google.com/blog/products/identity-security/find-and-fix-software-vulnerabilities-with-codemender/

## 核心主題
Google 推出 CodeMender，一個管理式程式碼安全代理，可自動掃描和修復軟體漏洞，利用 AI 技術加速安全生命週期。

## 關鍵重點
- **靈活的模型掃描**：CodeMender 提供多種模型選擇，可根據成本、速度和深度掃描性能進行優化，支援 C/C++、Go、Java、Python、Ruby、Rust 和 TypeScript 等程式語言。
- **自動修復漏洞**：通過模擬攻擊和執行概念驗證來驗證漏洞的可行性，優先修復最關鍵的漏洞，減少警報疲勞和誤報。
- **自動生成和測試程式碼修復**：自動生成安全修復补丁，使用 LLM 作為法官確保修復不會破壞現有應用程式功能，並可根據程式碼庫的編寫規範進行調整。

## 結論
CodeMender 是向連續、自我修復的代理軟體開發生命週期邁出的重要一步，其中程式碼在進入生產環境之前自動安全、驗證和修復，為開發團隊提供前所未有的速度和精度。
