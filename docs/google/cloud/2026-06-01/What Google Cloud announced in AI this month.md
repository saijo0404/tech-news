# Google Cloud 本月 AI 公告總覽：從 Gemini 3.5 到 Agentic Data Cloud

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-01
- **原文連結**: https://cloud.google.com/blog/products/ai-machine-learning/what-google-cloud-announced-in-ai-this-month/

## 核心主題
Google Cloud 發布 2026 年 1 至 6 月 AI 月度公告彙整，涵蓋 Gemini 3.5、Gemini Omni、Gemini Spark 等新世代模型發布、AI Hypercomputer 第八代 TPU、Gemini Enterprise Agent Platform 全面轉型、Agentic Data Cloud 架構、Wiz 併購完成，以及 Universal Commerce Protocol 等多項重磅創新，描繪企業 AI 從「對話式」邁向「代理式（Agentic）」的完整藍圖。

## 關鍵重點
- **模型與產品線全面升級（一月至六月）**：一月推出 Gemini Enterprise for CX 與 Universal Commerce Protocol（UCP）打造代理式商務基礎；二月發布 Gemini 3.1 Pro、Nano Banana 2、Claude 4.6/4.8 登陸 Vertex AI；三月推出 Gemini Embedding 2（首個原生多模態嵌入模型）、Veo 3.1 Lite 影片生成模型、Wiz 正式併入 Google Cloud、Gemini 3.1 Flash Live 語音 AI 突破 90.8% ComplexFuncBench 分數；Google I/O 2026 發表 Gemini 3.5（結合前沿智能與行動能力）、Gemini Omni（多模態輸入生成影片等多重輸出）、Gemini Spark（24/7 個人 AI 代理）、Google Antigravity 擴展與 Nano Banana 2/Pro GA；並推出 Google AI Threat Defense 自主安全平台。
- **AI Hypercomputer 第八代 TPU 與 Agentic Data Cloud 架構**：AI Hypercomputer 第八代 TPU 8t 專為訓練優化，採用突破性 ICI 技術可擴展至 9,600 顆 TPU 與 2PB 共享記憶體，處理能力提升三倍、每瓦效能提升兩倍；TPU 8i 專為推論優化，Boardfly 拓撲直接連接 1,152 顆 TPU，片上 SRAM 增加三倍、推論性價比提升 80%，支援百萬級平行代理。Agentic Data Cloud 以跨雲端 Lakehouse（標準化 Apache Iceberg）、Knowledge Catalog（統一動態情境圖表）與 Smart Storage Object Context API 打造 AI 原生資料架構，讓代理能即時感知、推理與行動。
- **Gemini Enterprise Agent Platform 轉型為企業代理核心**：自 Google Cloud Next '26 起，所有 Vertex AI 服務與路線圖轉型至 Agent Platform 統一交付；Gemini Enterprise App 新增 Agent Designer（無程式碼流程設計器）、長期代理（自主執行多步驟商業流程）、Inbox（中央監控中心）、Projects（受限記憶空間）、Skills（快速捷徑）、Canvas（互動編輯器）、Agent Gallery（第三方代理市集，含 Adobe、ServiceNow 等）；配合 Model Armor、Fraud Defense 代理安全防護、Wiz AI-BOM 開發工具整合，以及 GEAR（Gemini Enterprise Agent Ready）學習計畫，形成從開發、擴展、治理到優化的完整代理生命週期平台。

## 結論
從 January 的 UCP 代理式商務到 June 的 Gemini 3.5 與 Gemini Omni，Google Cloud 六個月內完成了從「單點 AI 工具」到「代理式生態系統」的全面轉型——模型層以 Gemini 3.5/Omni/Spark 覆蓋從推理到創作到自動化行動的全場景；基礎設施層以 AI Hypercomputer TPU 8t/8i 支撐規模化運算；資料層以 Agentic Data Cloud 與 Knowledge Catalog 讓代理能即時感知企業全域資料；安全層以 AI Threat Defense、Model Armor、Wiz 整合建構企業級防護；開發者層以 Agent Platform、Antigravity、GEAR 計畫降低代理建構門檻。這是一套完整的 Agentic Enterprise 藍圖，而 Google Cloud 正以月度節奏將藍圖逐一實現。
