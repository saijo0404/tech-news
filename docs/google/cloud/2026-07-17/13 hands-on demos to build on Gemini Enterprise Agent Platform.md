# 13 hands-on demos to build on Gemini Enterprise Agent Platform

- **來源**: Google Cloud Blog
- **發布日期**: 2026-07-18
- **原文連結**: https://cloud.google.com/blog/products/ai-machine-learning/13-demos-on-gemini-enterprise-agent-platform/

## 核心主題
Google 分享 13 個實作演示，展示如何在 Gemini Enterprise Agent Platform 上建立、擴展、治理和優化 AI 代理。

## 關鍵重點
- 使用 ADK (Agent Development Kit) 快速建立第一個代理，從基礎到進階的完整教學
- 建立事件驅動代理並引入人工介入（human-in-the-loop），處理超過閾值的費用申請
- 通過 Model Context Protocol (MCP) 連接代理到數據源，如 BigQuery、檔案搜索和 API 調用
- 使用 Agent-to-UI (A2UI) 建立動態前端，根據對話流實時組建用戶界面
- 將代理部署到 Agent Runtime 以處理生產流量，自動處理擴展和會話管理
- 建立可持久化的長運行代理，支持週日復和檢查點恢復
- 代理治理與安全，包括 STRIDE 威脅模型、Agent Gateway 和 Model Armor
- 代理質量優化飛輪，使用 AutoRaters 評估和優化代理表現
- 跨語言多代理管道，通過 A2A 協議連接 Python 和 Go 代理
- 跨框架代理擴展，整合 CrewAI、LangGraph、A2A 和 ADK 框架
- 完整的代理開發週期演示，從基礎到生產級代理的完整解決方案

## 結論
這些演示涵蓋了從基礎到生產級代理的完整開發週期，幫助開發者快速上手並構建企業級 AI 代理系統。通過 Agents CLI 和 ADK，開發者可以無需手動編寫大量代碼即可構建、部署和治理代理。
