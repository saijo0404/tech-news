# Claude Fable 5：現已上架 Google Cloud

- **來源**: Google Cloud Blog
- **發布日期**: 2026-06-09
- **原文連結**: https://cloud.google.com/blog/products/ai-machine-learning/cloud-fable-5-on-google-cloud/

## 核心主題
Anthropic 最新前緣模型 Claude Fable 5 正式於 Google Cloud 一般可用，作為 Google Agent Platform 持續引進產業最新模型的最新成果，支援複雜多步驟推理、進階軟體開發與深度多模態文件分析。

## 關鍵重點
- **Claude Fable 5 正式一般可用：Anthropic 最新前緣模型登上 Google Cloud Agent Platform**：Claude Fable 5 為 Anthropic 最新前緣模型，現已於 Google Cloud 正式一般可用（GA）；此發布展現 Google 持續承諾將產業最新模型直接提供至 Agent Platform；模型具備強大安全防護機制，確保適合一般使用，專門針對複雜多步驟推理任務設計。
- **適用高階任務：進階軟體開發、長程代理作業、深度多模態文件分析**：Claude Fable 5 適合處理高要求任務，包括進階軟體開發、長程代理（long-horizon agents）作業，以及深度多模態文件分析；詳細功能與發布資訊可參考 Anthropic 官方博客（mythos 5）。
- **Anthropic 模型陣容完整佈局：同步提供 Claude Opus 4.8 與 Sonnet 4.6**：Google Cloud Agent Platform 不僅提供 Claude Fable 5，同時提供 Anthropic 完整模型陣容，包括 Claude Opus 4.8 與 Claude Sonnet 4.6，開發者可依需求選擇不同效能與成本層級的模型進行建構。

## 結論
Claude Fable 5 上架 Google Cloud 標誌著 Anthropic 最新前緣模型透過 Google Agent Platform 正式開放企業與開發者使用——在複雜推理、軟體開發與多模態分析等高要求任務中提供業界領先能力，同時結合完善安全防護機制，為 Google Cloud 上的 AI Agent 應用提供強大模型後盾；加上 Claude Opus 4.8 與 Sonnet 4.6 的完整陣容，Google Cloud 已成為使用 Anthropic 模型進行 AI 開發的核心平台之一。
