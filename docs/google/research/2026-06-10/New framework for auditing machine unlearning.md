# New framework for auditing machine unlearning

- **來源**: Google Research Blog
- **發布日期**: 2026-06-10
- **原文連結**: https://research.google/blog/new-framework-for-auditing-machine-unlearning/

## 核心主題
Google 提出一套名為「正則化 f-散度核測試」的全新框架，用於更高效、更精確地審計 AI 模型是否真正「忘記」了指定的訓練資料。

## 關鍵重點
- **傳統兩樣本測試的缺陷**：現有常用的統計方法（如 MMD）只能偵測全域性的資料分佈偏移，難以發現局部且精細的隱私洩漏；同時，即使兩個模型用完全相同的資料重新訓練，也會因批次大小等差異產生假陽性，導致誤判。
- **提出相對距離三樣本測試**：框架引入「相對距離測試」，比較被消除資料的模型分佈更接近「安全重新訓練的模型」還是「原始含有敏感資料的模型」，並利用多種 f-散度（如 χ² 散度、KL 散度、冰球棍散度）作為靈敏的檢測器，自動選擇最合適的統計量與超參數設定。
- **實驗驗證優於現有方法**：在隱私審計中，冰球棍散度測試能以極少數樣本（幾千個而非數百萬個）有效偵測隱私違規；在機器消除評估中，傳統方法全數被標記為不合格，而新框架正確識別出安全模型，並發現目前常見的消除演算法（如微調、剪枝、選擇性突觸減弱）大多未能真正忘記目標資料。

## 結論
這套框架為審計 AI 模型行為提供了更精確、更靈活且具數學嚴謹性的工具，讓研究人員與審計者能夠以較少的樣本和手動調參，統計上證明模型是否安全、是否洩露了敏感資料。未來工作將著眼於進一步降低樣本複雜度並探索更優化的散度選擇。
