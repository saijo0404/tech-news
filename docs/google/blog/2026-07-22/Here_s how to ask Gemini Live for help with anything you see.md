# Here's how to ask Gemini Live for help with anything you see

- **來源**: Google Blog
- **發布日期**: 2026-07-22
- **原文連結**: https://blog.google/products-and-platforms/products/gemini/gemini-live-camera-how-to/

## 核心主題
這篇文章介紹了如何使用 Gemini Live 的相機功能，透過實時影像來獲得即時協助，無需冗長描述即可獲得專家協助。

## 關鍵重點
- 使用 Gemini app 的 Live 圖標並點擊相機，將鏡頭指向需要協助的物件（如複雜手冊、錯誤代碼或獨特物件）
- 可以請求逐步維修指導、整理雜亂抽屜的建議，或協助瀏覽複雜應用程式
- 還能用於創意靈感，如裝潢設計或購物建議

## 結論
Gemini Live 讓使用者透過實時影像獲得即時協助，將現實世界作為起點，解決各種視覺相關問題。
