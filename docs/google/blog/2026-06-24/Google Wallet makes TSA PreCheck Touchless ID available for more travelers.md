# Google Wallet makes TSA PreCheck Touchless ID available for more travelers

- **來源**: Google Blog
- **發布日期**: 2026-06-24
- **原文連結**: https://blog.google/products-and-platforms/platforms/google-pay/google-wallet-tsa/

## 核心主題
Google Wallet 成為 TSA PreCheck Touchless ID 計畫首個數位錢包合作夥伴，讓持有 TSA PreCheck 的旅客可透過手機數位護照與登機證，在 65 個機場快速完成面部驗證安檢，無需提供實體證件或登機證。

## 關鍵重點
- **從限縮到廣泛支援**：過去 TSA PreCheck Touchless ID 僅在 65 個機場對有限數量的航空公司開放，且旅客需為每家航空公司手動上傳護照資訊；整合 Google Wallet 後，所有 100 家 TSA PreCheck 合作航空公司旅客均可透過 Wallet 一鍵加入，流程簡化為：新增數位身份證（使用護照資訊）→ 登入航班並儲存登機證至 Wallet → 在登機證上看到「Get started」按鈕並導向 TSA 註冊頁面 → 同意安全分享數位身份證與登機證 → 確認後即可使用快速通道。
- **隱私與安全設計**：Google Wallet 強調隱私優先——只有在使用者明確選擇加入並透過生物辨識、PIN 或圖案解鎖裝置後，資訊才會分享給 TSA。數位身份證始終加密並直接儲存於手機中，使用者完全控制誰可讀取數位身份證內容。面部驗證由 TSA 的安全面部比對系統處理，無需提供實體身份證或登機證。
- **數位身份的持續擴展**：此次 TSA 合作是 Google Wallet 數位身分戰略的延續，之前已支援數位身份證、相片掃描添加票證等功能。透過持續整合各類數位憑證，Google Wallet 致力於讓數位生活更簡單且更安全。

## 結論
Google Wallet 與 TSA 的合作將原本繁瑣的 Touchless ID 註冊流程簡化為手機上的幾步操作，同時擴大支援航空公司至 100 家。在隱私安全設計（端到端加密、使用者自主控制、生物辨識認證）的基礎上，這項整合讓 TSA PreCheck 旅客的機場安檢體驗大幅升級，也展示了數位錢包在整合政府服務與提升公共設施效率上的潛力。
