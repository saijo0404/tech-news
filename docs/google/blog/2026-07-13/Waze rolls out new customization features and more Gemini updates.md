# Waze rolls out new customization features and more Gemini updates

- **來源**: Google Blog (Waze)
- **發布日期**: 2026-07-13
- **原文連結**: https://blog.google/waze/waze-updates-gemini-motorcycle-mode/

## 核心主題
Waze 推出新的自訂功能，包括機車模式、個人化導航、更少的語音提示以及 Gemini 能力整合，讓駕駛體驗更個人化與智能化。

## 關鍵重點
- **機車模式**：針對機車騎士提供專屬路線規劃，識別路緣、坑洞、速度坎等危險，已在阿根廷、巴西、哥倫比亞、馬來西亞、墨西哥、秘魯和菲律賓推出
- **個人化導航**：根據用戶過往行程建議路線，可選擇關閉此功能或選擇替代路線
- **少語音模式**：減少語音提示頻率，讓用戶更專注於音樂或播客，但仍保留關鍵安全提醒
- **會話式地圖更新**：用戶可以用自然語言報告道路狀況，如「這裡的路已封閉」，由當地地圖編輯員驗證並更新
- **Gemini 目的地搜尋**：用戶可以用語音詢問「幫我找附近開著的咖啡館」等問題，Waze 會提供選項列表

## 結論
Waze 透過這些更新讓導航更個人化、更智能，並整合 Gemini 能力提升用戶體驗，從自訂駕駛體驗到更新地圖資訊，讓旅程更貼合個人需求。
