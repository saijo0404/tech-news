# The DMA should not undercut security & privacy for Europeans

- **來源**: Google Blog
- **發布日期**: 2026-07-16
- **原文連結**: https://blog.google/company-news/inside-google/around-the-globe/google-europe/the-dma-should-not-undercut-security-privacy-for-europeans/

## 核心主題
Google 歐洲事務長肯特·沃克警告歐盟數位市場法案（DMA）若過度執行政策，可能削弱歐洲用戶的隱私與安全防護。

## 關鍵重點
- Android 規則可能允許外部應用程式獲取敏感設備權限，威脅裝置安全性
- 歐洲用戶的私人搜尋可能暴露給不熟悉的公司，缺乏足夠的匿名化處理
- 缺乏用戶知情同意，可能損害商業機密與國家安全

## 結論
Google 將繼續倡導平衡方法，在支持市場目標的同時保護隱私與安全。
