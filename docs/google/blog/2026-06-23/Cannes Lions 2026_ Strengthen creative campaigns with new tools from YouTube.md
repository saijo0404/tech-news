# Cannes Lions 2026: Strengthen creative campaigns with new tools from YouTube

- **來源**: Google Blog
- **發布日期**: 2026-06-23
- **原文連結**: https://blog.google/products/ads-commerce/youtube-insights-tools-cannes-lions-2026/

## 核心主題
Google 於坎城創意節發表一系列搭載 Gemini 的 YouTube 新洞察工具，旨在幫助品牌與代理商掌握創作者行銷趨勢、優化廣告創意效能，並強化在 YouTube 的付費與自然流量策略。

## 關鍵重點
- **更深入的趋势洞察**：Google Ads Insights Finder 新增美國市場趨勢洞察，提供更細緻的 YouTube 熱門內容分析，讓品牌即時掌握平台上的熱門話題與內容走向。
- **整合式品牌脈動報告**：部分 Brand Pulse Report 指標已整合至 Insights Finder，讓品牌能更清楚地檢視其在 YouTube 上的付費廣告與自然流量雙重表現，統一看板減少資料分散的問題。
- **Insights API 與 Demand Gen 創意優化**：新推出的 Content & Creator Insights API 提供關於 YouTube 創作者與受眾的 richer 數據，助代理商制定更有效率的媒體規劃。此外，Gemini 即將為 Demand Gen 活動提供創意洞察建議，例如推薦能提升表現的視覺元素。

## 結論
隨著 AI 推動創意行銷新時代的到來，Google 透過這些 Gemini 驅動的工具，讓品牌能更深入理解創作者影響力與內容趨勢。從趨勢洞察、品牌脈動到 API 與 AI 創意建議，YouTube 正逐步成為品牌在創作者行銷時代中不可或缺的決策支援平台。
