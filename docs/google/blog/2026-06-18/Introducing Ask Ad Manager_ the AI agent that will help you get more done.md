# Introducing Ask Ad Manager: the AI agent that will help you get more done

- **來源**: Google Blog
- **發布日期**: 2026-06-18
- **原文連結**: https://blog.google/products/admanager/ask-ad-manager/

## 核心主題
Google 推出 Ask Ad Manager——基於 Gemini 建構的多輪對話 AI 代理，幫助出版商快速排查問題、生成自訂效能報告，並透過自然語言互動提升 Google Ad Manager 平台的操作效率。

## 關鍵重點
- **即時排查問題**：出版商無需手動產生報告並篩選數據，只需透過對話即可診斷與解決線性項目（line item）問題，並可追問以識別阻礙營收的因素，加速問題解決。
- **一鍵生成自訂報告**：當需要特定指標分析庫存表現時，出版商只需輸入提示（prompt），Ask Ad Manager 即可自動產生報告、表格或比較基準，快速提供深入洞察，取代手動整合多個報表的繁瑣過程。
- **智慧導覽與開發者工具**：Ask Ad Manager 會根據對話脈絡提供個人化指引與精確連結，直接導航至 Ad Manager 中的正確設定頁面，並自動載入適當的篩選條件。Yahoo 等出版商已開始將 Ad Manager 整合至自訂代理中；Google 預計今年晚些時候推出 REST API 與 MCP Server 等新開發者工具，並發展專用於出版商與廣告代理的專屬代理。

## 結論
Ask Ad Manager 將 Google Ad Manager 從手動操作工具轉型為 AI 驅動的代理式平台——從排查、報告到導覽，出版商能以對話方式完成以往耗時的任務，而即將推出的開發者工具與專屬代理生態系，更將推動廣告技術產業邁向 AI 自主操作的下一個階段。
