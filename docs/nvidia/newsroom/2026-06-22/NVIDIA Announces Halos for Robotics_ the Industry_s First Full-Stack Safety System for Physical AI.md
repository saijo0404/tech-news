# NVIDIA Announces Halos for Robotics: the Industry's First Full-Stack Safety System for Physical AI

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-06-22
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-announces-halos-for-robotics-the-industrys-first-full-stack-safety-system-for-physical-ai

## 核心主題
NVIDIA 宣布推出 Halos for Robotics——全球首個完整的開放式機器人安全系統，將 NVIDIA Halos 在自動駕駛領域的驗證安全基礎延伸至工業機器人與人型機器人，為在現實世界中感知、決策與行動的 AI 系統提供單一共通的安全架構。

## 關鍵重點
- **三層級全堆疊安全架構**：（1）硬體層——IGX Thor 與 Holoscan Sensor Bridge 提供工業級 AI 運算、內建安全功能與感測器串連，支援即時機器人與安全工作負載；（2）軟體層——Halos OS 包含 Halos Core（安全相關作業功能）與 Outside-In Safety Blueprint（利用外部攝影機與 AI Agent 擴展機器人感知並動態控制行為）；（3）認證層——Halos AI Systems Inspection Lab 是全球首個獲 ANAB 認證的功能安全與 AI 安全檢驗計畫，協助夥伴為向 TÜV、UL、SGS 等機構申請第三方認證做準備。
- **Agility 為首個整合夥伴**：人型機器人領導廠商 Agility 將 IGX Thor 與 Halos Core 整合至其 Digit 人型機器人的自有人類檢測系統，服務 Amazon、Schaeffler、Toyota Motor Canada 等客戶的工廠、倉儲與物流作業。Agility 同時參與 Halos AI Systems Inspection Lab，確保 Digit 的安全軟體、AI 元件與網路安全防護符合 IEC 61508、ISO 13849 與 ISO/IEC TR 5469 等嚴苛標準。
- **跨生態系合作與即早存取**：生態系涵蓋軟體（Acontis、FreeRTOS、QNX）、嵌入式系統（Advantech、NexCobot）、感測器與晶片（Infineon、NXP、ST、TI）、工業應用（FORT Robotics、KION Group、Lyte AI、Neurealm）與檢驗機構。Halos Core 已對註冊開發者開放 Linux 與 Linux+QNX 兩種組態的早期存取；Outside-In Safety Blueprint 已於 GitHub 開放原始碼早期存取。

## 結論
NVIDIA Halos for Robotics 將自動駕駛領域超過 18,600 年工程驗證的安全基礎標準化並移植至物理 AI 領域，解決了人型機器人在動態工業環境中與人類共同作業時的安全挑戰——從硬體安全島、認證作業系統到第三方檢驗認證路徑，構成完整的端到端安全生態系。Agility 的首個採用證明了產業已準備好從臨時安全實作邁向共享、國際標準化的安全架構，讓機器人能以更高信心進駐工廠與物流環境。
