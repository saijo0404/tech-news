# 企業軟體領導廠商攜手 NVIDIA 打造 AI 代理

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-05-31
- **原文連結**: https://nvidianews.nvidia.com/news/enterprise-software-leaders-build-ai-agents-with-nvidia

## 核心主題
NVIDIA 推出 Agent Toolkit 全新軟體、開源模型與夥伴合作計畫，以 Nemotron 開源模型、NemoClaw 藍圖、OpenShell 安全執行環境與 CUDA-X 函式庫技能，讓企業軟體領導廠商建構可長途運行的自主 AI 代理，將工程與營運流程從數週縮短至數小時。

## 關鍵重點
- **CAD 與工業設計龍頭建構自主 AI 工程師**：Cadence、Dassault Systèmes、Siemens、Synopsys 等率先使用 NemoClaw 打造自主 AI 工程師，將晶片設計、仿真驗證等耗時工作壓縮至數小時；Cadence 的 ChipStack 成為首款自主 AI 晶片驗證代理，Siemens 的 Fuse EDA 代理跨半導體與 3D IC 設計協調多工具流程。
- **Nemotron 3 Ultra 模型與安全生態系擴展**：5500 億參數 MoE 模型提供高達 5 倍推理加速與 30% 成本降低，已微調支援 Hermes Agent、OpenClaw、OpenHands 等平台；CrowdStrike 與 Palantir 運用開源模型打造長途運行的安全與企業代理；OpenShell 安全執行環境整合至 Microsoft Windows、Ubuntu、Red Hat、SAP 與 ServiceNow，支援從個人裝置到資料中心的全面部署。
- **CUDA-X 函式庫轉為代理技能**：cuDF、cuOpt、AI-Q、NeMo、PhysicsNeMo、CUDA-Q 等函式庫現以領域技能形式開放給 AI 代理使用，使代理能直接存取高效能資料處理、最佳化路由、量子計算模擬等專門能力。

## 結論
NVIDIA 透過 Agent Toolkit 打造從模型、藍圖、安全執行環境到運算函式庫的完整生態系，讓企業軟體廠商能安全、可擴展地部署自主 AI 代理，重新定義工程、安全、營運等領域的工作方式。
