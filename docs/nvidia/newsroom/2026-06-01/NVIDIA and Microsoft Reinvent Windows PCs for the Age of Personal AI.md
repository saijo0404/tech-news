# NVIDIA 與 Microsoft 重塑 Windows PC 迎向個人 AI 時代

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-05-31
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-microsoft-windows-pcs-agents-rtx-spark

## 核心主題
NVIDIA 發布 RTX Spark 超級晶片，攜手 Microsoft 推出首款為個人代理打造的 Windows 電腦，以 1 petaflop AI 運算、128GB 統一記憶體與 Windows 原生安全機制，將電腦從工具轉變為夥伴。

## 關鍵重點
- **RTX Spark 超級晶片規格與生態系**：搭載 Blackwell RTX GPU（6,144 CUDA 核心、第五代 Tensor Cores）與 20 核心 Grace CPU（由 MediaTek 協同設計），透過 NVLink-C2C 互連，支援 90GB+ 3D 場景渲染、12K 影片編輯、120B 參數 LLM 與 100 萬 token 上下文、1440p 100+ FPS 遊戲，並導入 DLSS 4.5 光線重建與 RTX Video 4 倍幀產生。
- **Windows 原生代理安全架構**：Microsoft 新安全原語（身分、隔離、策略、端對端安全）與 NVIDIA OpenShell 執行環境結合，確保代理在本機安全運行；OpenClaw 與 Hermes Agent 將整合至全新 Windows 應用程式，並從 Windows 任務列直接存取代理功能。
- **Adobe 重新架構創意工具與硬體夥伴集結**：Adobe 全面重構 Photoshop 與 Premiere 以運用統一記憶體與 TensorRT，AI 編輯與特效處理速度提升最高 2 倍；ASUS、Dell、HP、Lenovo、Microsoft Surface、MSI、Acer 與 GIGABYTE 將於今年秋季推出 14 至 16 吋筆電與緊湊型桌上型機，最薄僅 14 毫米、重 3 磅。

## 結論
RTX Spark 結合 NVIDIA 全堆疊技術與 Microsoft Windows 安全架構，打造全球首款個人 AI 電腦平台，讓個人代理、創意工作流與 AAA 遊戲都能在同一台筆電上運行，正式開啟「個人 AI 電腦」新紀元。
