# NVIDIA 發布 Vera：首款為代理設計的 CPU

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-05-31
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-unveils-vera-the-cpu-for-agents

## 核心主題
NVIDIA 宣布量產首款專為代理式 AI 設計的 Vera CPU，以自研 Olympus 核心、88 核心架構與 LPDDR5X 記憶體子系統，提供比 x86 處理器快 1.8 倍的任務完成速度，將 AI 工廠的衡量指標從「每美元核心數」轉變為「每美元 token 產量」。

## 關鍵重點
- **Olympus 核心驅動代理工作負載**：Vera 搭載 88 個 Olympus 核心與 1.2 TB/s 記憶體頻寬，支援 Python 執行、沙盒程式碼、編譯、資料處理與協調邏輯；Phoronix 開放基準測試驗證其在程式編譯、Python、Java 與資料庫處理等代理關鍵路徑工作負載中表現最快。
- **廣泛生態系採用**：Anthropic、OpenAI、SpaceXAI 評估導入 Vera 擴展 CPU 密集型代理工作負載；Oracle、ByteDance、CoreWeave 等超大型雲服務商規劃部署；Dell、HPE、Lenovo、Supermicro 等硬體廠將推出獨立 Vera CPU 伺服器，成為 x86 之外的首個標準 CPU 選項。
- **全 AI 工廠部署與安全整合**：Vera 透過 NVLink-C2C 互連（高達 1.8 TB/s CPU-GPU 頻寬）擔任 Vera Rubin 平台主機 CPU，支援液冷密集機架與空冷雙槽系統；Vera BlueField-4 STX 整合網路、儲存加速與晶片內安全，建構安全由設計的 AI 原生資料平台。

## 結論
Vera CPU 以代理優先的架構設計，結合高單核心效能、高併發執行與省電頻寬，讓代理能更快完成多步驟任務、加速器不必等待排程與資料移動，為代理式 AI 與強化學習時代打造新一代資料中心運算基礎。
