# NVIDIA 釋出重大開源代理工具與技能集合，推動實體 AI 開發

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-05-31
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-releases-major-collection-of-open-source-agent-tools-and-skills-for-physical-ai

## 核心主題
NVIDIA 宣布開放一整套實體 AI（Physical AI）代理工具與技能（Skills），涵蓋 Omniverse、Cosmos、Alpamayo 與 Metropolis 等平台，讓開發者能將複雜的機器人、自動駕駛、視覺 AI 與工業數位孿生流程轉換為代理可執行的任務，大幅降低規模化開發的成本與複雜度。

## 關鍵重點
- **代理友善的實體 AI 工具鏈**：NVIDIA 將 Cosmos 世界基礎模型、Omniverse 模擬與數位孿生、Isaac 機器人學習、Metropolis 視覺 AI、Alpamayo 自動駕駛及 Jetson 邊緣平台整合物流，透過 NVIDIA Agent Toolkit 與 NemoClaw 藍圖，使 AI 代理能直接呼叫 NVIDIA 函式庫與模型，自動化資料生成、模擬、訓練與部署。
- **跨產業加速應用**：台積電與和碩利用缺陷影像生成技能縮短模型訓練部署時間 67%；緯創降低 30% 缺陷資料收集量；鴻海提升一次良率 3%；理想汽車、阿法智與深之藍使用 Omniverse NuRec 每日生成 1,000 以上重建與 30 萬張渲染模擬。
- **醫療與工業數位孿生拓展**：Foxconn 的 Nurabot 護理機器人已擴展至多家醫院；台達電利用合成缺陷資料提升銲點檢測率 17%；Cadence、Siemens、Synopsys 等工業軟體龍頭運用 Omniverse 建構工程模擬與互動數位孿生；SK 海力士推動 Autonomous Fab 2030 計畫。

## 結論
NVIDIA 將實體 AI 工具全面代理化，透過 GitHub 與 skills.sh 免費開放，並與 Microsoft、CoreWeave、Nebius 等雲端服務商整合，加速機器人、自動駕駛、醫療照護與智慧製造等領域的實體 AI 創新與部署。
