# NVIDIA Agent Toolkit Expands With New Omniverse Libraries_ Putting AI Agents to Work Building Simulation-Ready Worlds

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-07-20
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-agent-toolkit-expands-with-new-omniverse-libraries-putting-ai-agents-to-work-building-simulation-ready-worlds

## 核心主題
NVIDIA Agent Toolkit 新增 Omniverse 軟體套件，賦予 AI 代理工具與技能，協助開發者將物理 AI 能力整合至現有應用程式，並為機器人、工廠與自主系統建立模擬準備好的環境。

## 關鍵重點
- **Omniverse 庫功能**：新增 ovrtx（RTX 感測器模擬）、ovphysx（GPU 加速物理模擬）及 CAD-to-SimReady 技能，可公開在 GitHub 上獲取
- **合作夥伴整合**：SideFX、PTC 等軟體廠商正將 Omniverse 庫整合至 Houdini、Onshape 等 3D 應用程式中，支援雲端與本地 AI 系統
- **Blender 藍圖發布**：提供模擬準備的 Blender 工作流藍圖，可從 RTX Spark 到 DGX Station 系統運行

## 結論
NVIDIA 透過 Agent Toolkit 與 Omniverse 庫的結合，推動「物理 AI 時代」的到來，讓開發者能在現實世界操作前，透過模擬環境訓練與測試機器人、工廠與自主系統，加速從 3D 內容到模擬準備環境的過渡。
