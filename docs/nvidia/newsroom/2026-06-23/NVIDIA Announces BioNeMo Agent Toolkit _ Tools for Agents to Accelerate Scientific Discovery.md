# NVIDIA Announces BioNeMo Agent Toolkit: Tools for Agents to Accelerate Scientific Discovery

- **來源**: NVIDIA Newsroom
- **發布日期**: 2026-06-23
- **原文連結**: https://nvidianews.nvidia.com/news/nvidia-launches-bionemo-agent-toolkit-giving-ai-agents-the-tools-to-accelerate-scientific-discovery

## 核心主題
NVIDIA 發布 BioNeMo Agent Toolkit——整合超過十年生物分子運算技術，提供領域專屬工具與技能，將通用 AI Agent 轉化為具備博士級研究助理能力的生命科學 AI 科學家，加速蛋白質設計、分子生成、基因組分析與藥物發現等研究循環。

## 關鍵重點
- **前沿模型 × 科學工具箱 = AI 科學家的崛起**：黃仁勳將此定位為「前沿模型是大腦，BioNeMo 是科學工具箱」——Nemotron 提供推理基礎，BioNeMo 提供加速生物分子工具（NIM 微服務、Parabricks、NemoClaw 藍圖、OpenShell 執行環境），讓 Agent 能理解科學知識、使用科學工具並執行科學工作流。全球 50 多家企業（Dassault Systèmes、Databricks、Lilly、Schrödinger、Snowflake 等）與 Anthropic、OpenAI 已開始採用，將通用 Agent 轉化為能完成蛋白質結構預測、分子对接、生成化學、基因組分析、蛋白質設計與生物標記發現的生命科學 Agent。
- **五大科學工作流應用場景**：（1）虛擬篩選——Agent 生成候選化合物、对接目標蛋白、預測結合強度並過濾藥物特性，將篩選時間從數天壓縮至數分鐘；（2）基因組分析與目標發現——Parabricks 加速對齊與變異呼叫，基因組基礎模型評分變異效應，Agent 排序最相關疾病候選；（3）蛋白質結合劑設計——Agent 在實體實驗前完成計算設計與驗證；（4）深層生物醫學研究——Biomedical AI-Q Research Agent 整合真實世界數據與推理模型，加速文獻回顧、方案生成、臨床試驗篩選與藥物警戒；（5）醫學影像分析——處理、分割、合成醫學影像數據以支援生物標記發現。
- **跨產業生態系共建**：科技平台（Benchling、Certara、Databricks、Snowflake、Seqera）連接數據系統與 AI 科學；診斷與製藥（Lilly、Natera）擴展可重複 Agent 工作流；AI 原生生物公司（Boltz、Basecamp Research、Chai Discovery、Dyno、PerturbAI、Proxima）協作開發模型驅動的治療設計工具；計算輔助藥物發現軟體（Dassault Systèmes、Cadence/OpenEye、Schrödinger）將工具整合至科學應用程式；實驗室儀器自動化（Automata、HighRes、Tecan、Thermo Fisher、Medra）連接實體系統與計算發現；AI 雲與基礎設施（Baseten、Modal、Nebius）以 API 與可擴充運算支援 Agent 生物學工作流。華盛頓大學蛋白質設計研究所（IPD）合作加速 RosettaFold3 等模型，效能提升 2 倍。

## 結論
BioNeMo Agent Toolkit 代表了科學研究方法的根本轉變——從「單次發現」邁向「迭代速度」，讓 AI Agent 以人類無法企及的速度重複推理生物複雜性。NVIDIA 將超過十年的生物分子運算堆疊轉化為 Agent 可呼叫的技能，不再只是提供模型，而是提供「能完成科學工作」的完整工具鏈。從虛擬篩選數天至數分鐘的壓縮，到 50 多家企業與研究機構的生態系共建，BioNeMo 正開啟一個 AI Agent 與人類科學家協同加速發現的新時代。
