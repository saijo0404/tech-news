# NVIDIA Vera CPU Opens the Way for Agentic Scientific AI at Los Alamos National Laboratory

- **來源**: NVIDIA Blogs
- **發布日期**: 2026-06-22
- **原文連結**: https://blogs.nvidia.com/blog/nvidia-vera-cpu-los-alamos-national-laboratory/

## 核心主題
洛斯阿拉莫斯國家實驗室（LANL）宣布與 HPE 和 NVIDIA 合作建構三台全新超算「Mission、Vision 與 Veritas」，全面採用 NVIDIA Vera CPU 與 Rubin GPU 平台，為科學研究導入具備自主提出假說、執行模擬與分析結果能力的 AI Agent。

## 關鍵重點
- **三台全新超算架構與部署**：三台系統皆採用 HPE Cray Supercomputing GX5000 架構與 NVIDIA Vera Rubin 平台（Vera CPU、Rubin GPU、Quantum-X800 InfiniBand 網路）。其中 Mission 包含 NVIDIA Vera Rubin GPU 節點與 2,300 顆獨立 Vera CPU（HPE Cray GX240 刀片）；Veritas 配置約 1,150 顆獨立 Vera CPU，服務實驗室導向研發計劃（LDRD），並測試這些技術於未來更大規模系統的應用；Vision 則服務基礎科學研究。Mission 預計 2027 年營運，將取代 Crossroads 處理機密國家安全工作負載。
- **Vera CPU 在 URSA Agent 與模擬工作負載上展現顯著效能**：LANL 的 URSA（Universal Research and Scientific Agent）——一模組化、回饋驅動的 AI 框架，能協助科學家腦力激盪假說、規劃實驗、執行模擬與分析結果——在 Vera CPU 上展現比 Crossroads x86 超算 CPUs 高出 7 倍的效能。在 Branson 蒙特卡羅熱傳模擬測試中，Vera CPU 表現超過 x86 CPU 3 倍以上，得益於其客製化 Olympus 核心、LPDDR5 記憶體與高速晶片內互連。單顆 Vera CPU 效能超過單插槽 x86 CPU 3 倍以上，且每核心記憶體提供 4 倍以上、每節點記憶體提供 6 倍以上。
- **深度共同設計傳統與 LANL-NVIDIA 十年合作**：所有實驗室超算皆由硬體架構師、系統軟體開發者、領域科學家、電腦科學家與應用數學家共同設計，確保系統由真實科學工作負載塑造而非僅依賴抽象基準測試。此合作延續 LANL 與 NVIDIA 超過十年的 CPU 協同設計傳統（從 Grace 到 Vera），Venado（2024 年安裝，搭載 GH200 Grace Hopper Superchips）為基礎擴展。

## 結論
NVIDIA Vera CPU 與 Rubin 平台的結合，不僅為 LANL 的下一代超算提供強大的計算基礎，更開啟了「代理式科學 AI」的新篇章——AI 不再只是輔助工具，而是能自主形成假說、選擇工具、啟動模擬並分析輸出的研究夥伴。這三位一體的新超算系統（2027 年全面營運）將使 LANL 的科學發現速度與規模達到前所未有的水準。
