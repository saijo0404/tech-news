# NVIDIA 與 Microsoft 合作打造統一堆疊，推動代理式 AI 從 Windows 裝置到雲端與本地部署

- **來源**: NVIDIA Blog
- **發布日期**: 2026-06-02
- **原文連結**: https://blogs.nvidia.com/blog/microsoft-build-windows-local-cloud-devices/

## 核心主題
NVIDIA 與 Microsoft 在 Microsoft Build 宣布擴大合作，整合 RTX Spark、DGX Station for Windows、NVIDIA 開源模型與 OpenShell 安全執行環境，打造從 Windows 個人裝置到 Azure 雲端與本地資料中心的代理式 AI 統一堆疊。

## 關鍵重點
- **Windows 裝置重塑與企業級代理部署**：RTX Spark 筆電與小型桌上型機以 1 petaflop AI 運算、128GB 統一記憶體打造全球首款個人代理用 Windows PC；DGX Station for Windows 搭載 GB300 Grace Blackwell Ultra 超級晶片、748GB 一致記憶體與 20 petaflops FP4 效能，支援高達 1 兆參數的前緣模型；兩者皆以 NVIDIA OpenShell 提供安全由設計的代理執行環境。
- **Microsoft Foundry 整合 NVIDIA 開源模型與資料倉儲加速**：Nemotron 3 Ultra、Anthropic Claude 模型（原生運行於 Azure 上的 GB300 系統）、Cosmos 3 物理 AI 模型與 Earth-2 天氣模型全部上架 Foundry 管理運算；Microsoft Fabric Data Warehouse 內建 NVIDIA 加速運算，SQL 執行速度比 CPU 基線快 6 倍、比其他雲端資料倉儲快 7 倍。
- **本地化 AI 部署與 Fairwater 資料中心驗證**：Foundry Local 在 Azure Local 上支援 RTX PRO 6000 Blackwell Server Edition，企業可在本地或主權環境部署高性能 AI 工作負載；Microsoft Fairwater Wisconsin AI 工廠提前上線，數萬套 Grace Blackwell 系統整合為單一 AI 工廠，並已驗證 Vera Rubin 平台可在 Azure 資料中心無改造部署，提供 10 倍每百萬瓦推論吞吐量提升。

## 結論
NVIDIA 與 Microsoft 透過硬體、模型、安全執行環境與雲端/本地運算的完整整合，讓代理式 AI 從開發者桌面延伸至企業資料中心，同時確保安全、治理與成本效益，為代理式 AI 的大規模生產部署奠定基礎。
