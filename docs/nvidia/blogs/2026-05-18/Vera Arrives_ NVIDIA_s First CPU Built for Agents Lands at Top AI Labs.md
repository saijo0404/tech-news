# Vera Arrives: NVIDIA’s First CPU Built for Agents Lands at Top AI Labs

- **來源**: blogs  
- **發布日期**: 2026-05-18  
- **原文連結**: https://blogs.nvidia.com/blog/vera-cpu-delivery/  

## 核心主題  
NVIDIA推出首款針對代理AI設計的Vera CPU，已交付至Anthropic、OpenAI、Oracle等頂級AI實驗室，標誌著代理AI基礎設施的實質突破。  

## 關鍵重點  
- Vera是NVIDIA首顆專為代理AI設計的CPU，具備88個自訂核心、1.2TB/s記憶體帶寬與50%更快的單核效能。  
- 首批Vera CPU已送達Anthropic、OpenAI、SpaceXAI與Oracle Cloud Infrastructure，支援代理AI的協調、工具呼叫與大規模推理工作負載。  
- Oracle計畫2026年起部署數十萬顆Vera CPU，強調其在企業級AI基礎設施的規模與效率優勢。  

## 結論  
Vera的推出代表NVIDIA在代理AI硬體領域的關鍵里程碑，透過專屬架構提升AI工廠效率，並加速企業與雲端服務商的應用部署。
