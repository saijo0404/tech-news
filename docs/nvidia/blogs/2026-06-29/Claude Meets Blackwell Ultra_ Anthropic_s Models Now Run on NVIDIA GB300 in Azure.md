# Claude Meets Blackwell Ultra: Anthropic's Models Now Run on NVIDIA GB300 in Azure

- **來源**: NVIDIA Blogs
- **發布日期**: 2026-06-29
- **原文連結**: https://blogs.nvidia.com/blog/anthropic-nvidia-gb300-blackwell-ultra-microsoft-azure/

## 核心主題
Anthropic Claude 模型透過 Microsoft Foundry 在 Azure 上正式全面啟用，運行於 NVIDIA GB300 Blackwell Ultra GPU 與 Quantum-X800 InfiniBand 網路，為企業提供建置自主與領域專屬 AI 智能體的強大運算基礎。

## 關鍵重點
- **GB300 Blackwell Ultra 運算加速**：Claude in Foundry 搭載 NVIDIA GB300 NVL72 系統與 Quantum-X800 InfiniBand 網路，使企業能建置與運行更強大的自主智能體系統，包括跨業務領域運作的專性子智能體，執行進階任務。優異的推理效能與效率降低了總持有成本。
- **NVIDIA 工具整合與驗證智能體技能**：NVIDIA 與 Anthropic 合作將 NVIDIA 工具整合至 Anthropic 堆疊，企業可透過 NVIDIA 驗證智能體技能（NVIDIA verified agent skills）賦予 Claude 智能體領域專屬能力，將 AI 智能體深植企業營運體系。
- **安全治理與基礎設施級管控**：企業可透過 NVIDIA Secure Agent Workspace Reference Design 在 Azure 上運行 Claude 智能體，提供受治理環境的藍圖，在基礎設施層級控制身份、網路存取、憑證與執行時期政策。此合作源自 2025 年 11 月 Microsoft、NVIDIA 與 Anthropic 的戰略夥伴關係。

## 結論
Claude 模型在 GB300 Blackwell Ultra 上的部署標誌著企業 AI 智能體運行的新里程碑——結合 Claude 的推理能力、NVIDIA 的硬體加速與 Azure 的雲端基礎設施，讓企業能安全、高效地部署自主智能體系統。透過 NVIDIA 工具鏈的深度整合與基礎設施級治理藍圖，AI 智能體正從實驗性工具轉變為企業營運的核心操作系統。
