# License to Stream: ‘007 First Light’ Coming to GeForce NOW With an Ultimate Bundle

- **來源**: blogs  
- **發布日期**: 2026-05-21  
- **原文連結**: https://blogs.nvidia.com/blog/geforce-now-thursday-007-first-light-ultimate-bundle/

## 核心主題  
NVIDIA GeForce NOW 推出《007 First Light》終極會員套件，玩家可透過訂閱獲得遊戲並即時串流體驗。

## 關鍵重點
- 007 First Light 終極會員套件包含新遊戲《007 First Light》，玩家可於遊戲發售當日立即遊玩。  
- 遊戲以間諜動作冒險為主，結合 cinematic 視覺效果與全球間諜任務，呈現詹姆斯龐德的重構起源故事。  
- 同期新增多款遊戲至 GeForce NOW，包括《Forza Horizon 6》與其他新作，強調雲端串流無需下載的便利性。

## 結論  
NVIDIA 透過終極會員套件與雲遊戲服務，提供高畫質遊戲體驗，降低硬體門檻並擴展玩家遊戲選擇。
