# AI 工廠：智慧時代的新基礎建設

- **來源**: NVIDIA Blog
- **發布日期**: 2026-05-27
- **原文連結**: https://blogs.nvidia.com/blog/ai-factories-the-new-infrastructure-of-intelligence/

## 核心主題
AI 工廠是專為持續、即時生產智慧而設計的新型基礎建設，將能源轉換為 token——推理模型、代理與智能系統的產量單位，成為 AI 時代的核心基礎設施。

## 關鍵重點
- **AI 工廠的經濟模型**：以每秒 token 數、每瓦 token 數、每個 token 成本、利用率與可用時間來定義經濟效益；每瓦效能直接轉化為收入，每個 token 成本影響所有 AI 工廠的經濟結構。
- **代理 AI 改變工作負載**：自主代理不僅回應提示，還進行推理、規劃、搜尋、使用工具、檢索數據、編寫程式與採取行動，甚至建立子代理，使 AI 工作負載更長、更深、更耗計算資源。
- **全堆疊協同設計**：從加速運算、高速記憶體、網路、儲存到軟體，每一層都持續最佳化以提升利用率、降低成本並提高產量；Blackwell Ultra GPU 帶來最低 token 成本，GB300 NVL72 系統每百萬瓦生成 50 倍於前代的 token，成本降低 35 倍。

## 結論
AI 工廠已從單純的 GPU 擴展為包含加速運算、高速互連、液冷系統、推理軟體與自主代理的全堆疊基礎建設，將能源轉換為智慧，成為推動下一波經濟成長的核心基礎設施。
