# Cosmos 3 如何讓實體 AI 在行動前先思考

- **來源**: NVIDIA Blog
- **發布日期**: 2026-05-31
- **原文連結**: https://blogs.nvidia.com/blog/cosmos-3-physical-ai-open-world-foundation-model/

## 核心主題
NVIDIA 發布 Cosmos 3 開放世界基礎模型，整合視覺推理、多模態生成與動作預測於單一架構，讓機器人、自動駕駛與視覺 AI 系統能在真實世界中理解場景因果並預測下一步，實現「行動前先思考」。

## 關鍵重點
- **OmniModel 原生動作生成**：Cosmos 3 以混合 Transformer 架構將推理區塊與生成區塊結合，不僅能理解場景，還能直接輸出機械臂關節角度、夾爪位置與軌跡點等數值化動作數據，開發者可針對特定機器人本體、相機佈局與任務進行微調。
- **智慧城市與空間推理**：Cosmos 3 能辨識場景中移動物體、路徑交會點與未來可能狀態，生成密集描述與預測場景變化；Linker Vision 已運用其視覺語言推理能力分析即時鏡頭串流，進行空間理解與根本原因分析。
- **長尾稀有場景生成**：針對碰撞與極端邊緣案例難以在真實世界安全重複捕捉的痛點，Cosmos 3 作為視頻基礎模型可生成物理上合理的情境序列，支援合成數據流程與未來狀態預測；Cosmos 3 變體在 Artificial Analysis、Physics-IQ、R-Bench 等多個世界生成基準測試中排名第一。

## 結論
Cosmos 3 以單一模型統一了感知、預測與動作生成，搭配 OpenMDW 1.1 開放授權，讓開發者能透過 NVIDIA NIM 微服務與 Hugging Face 快速部署，推動實體 AI 從實驗室走向可擴展的真實世界應用。
