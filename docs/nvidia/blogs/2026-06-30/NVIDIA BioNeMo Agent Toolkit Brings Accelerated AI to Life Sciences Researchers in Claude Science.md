# NVIDIA BioNeMo Agent Toolkit Brings Accelerated AI to Life Sciences Researchers in Claude Science

- **來源**: NVIDIA Blog
- **發布日期**: 2026-06-30
- **原文連結**: https://blogs.nvidia.com/blog/claude-science-bionemo-agent-toolkit/

## 核心主題
NVIDIA 宣布 BioNeMo Agent Toolkit 與 Anthropic Claude Science 整合，讓生命科學研究人員能以自然語言與 AI 代理對話，自動調用 GPU 加速的科學工作流程，加速藥物發現、基因體學和蛋白質工程研究。

## 關鍵重點
- **自然語言驅動科學研究**：Claude Science 讓科學家以自然語言描述研究任務（如分析基因體序列、預測蛋白質結構、設計抑制劑），AI 代理自動解析請求並調用預先配置的領域專業代理，無需手動設定模型或軟體環境。整合 BioNeMo Agent Toolkit 後，代理可選擇合適的 NVIDIA 加速工具、準備輸入並執行工作流程。
- **極致加速的科學工具套件**：BioNeMo Agent Toolkit 將 NVIDIA 加速能力封裝為可調用的 skills：Parabricks 將基因體分析從數小時縮短至數分鐘；RAPIDS-singlecell 將 130 萬細胞的處理與聚類從 52 分鐘縮至 25 秒；nvMolKit 將化學資訊學操作加速達 **3,000 倍**；並提供 Evo 2、Boltz-2、OpenFold3 等開源模型與 NIM 微服務。
- **科學推理與加速運算的迭代循環**：研究人員與 AI 代理形成迭代迴圈——代理執行加速運算，研究人員審查輸出、修正問題並決定下一步，全程專注於科學本身。例如設計癌症抑制劑時，代理可自動完成高通量抑制劑預測、最佳化與驗證。目前全球前 20 大藥廠中有 18 家使用 NVIDIA BioNeMo。

## 結論
BioNeMo Agent Toolkit 將 NVIDIA 十年的 GPU 加速積累以「可調用 skills」的形式注入 AI 代理生態，讓 Claude Science 成為科學家的 AI 研究助理。這不僅加速單一計算任務，更將加速運算融入科學推理的每一步，標誌著生命科學研究邁向「代理時代」的重要一步。
