# NVIDIA Confidential Computing to Help Expand Apple's Private Cloud Compute

- **來源**: NVIDIA Blog
- **發布日期**: 2026-06-09
- **原文連結**: https://blogs.nvidia.com/blog/nvidia-confidential-computing-apple-private-cloud-compute/

## 核心主題
NVIDIA 宣布其搭載保密計算（Confidential Computing）功能的 GPU 已正式納入 Apple Private Cloud Compute（PCC），在擴展至 Google Cloud 的同時，為 Apple Intelligence 提供硬體級別的隱私保護與安全推論能力。

## 關鍵重點
- **Apple × NVIDIA × Google 三方合作**：NVIDIA Blackwell GPU 將支援 Apple Foundation Models（由 Apple 與 Google 共同建構、基於 Gemini 系列技術）的伺服器端推論，PCC 架構從 Apple 自家資料中心擴展至 Google Cloud，並整合保密計算作為硬體安全架構的一環。
- **保密計算的硬體級隱私保護**：透過可信執行環境（TEE）隔離工作負載、加密通訊路徑、遠端驗證機制，確保即使系統建造者也无法查看使用者的資料、聊天或對話內容，同時維持 GPU 的高效能運算能力。
- **AI 基礎設施的新趨勢**：隨著 AI 體驗結合裝置端與雲端處理，高效能伺服器端推論與嚴格隱私安全保證的結合變得日益重要，NVIDIA 保密計算技術為處理敏感資訊的 AI 服務提供了可靠的基礎設施選擇。

## 結論
NVIDIA 與 Apple、Google 的合作標誌著 AI 基礎設施從「效能優先」轉向「效能與隱私並重」的重要里程碑，透過硬體層級的保密計算，讓大規模 AI 服務能在不犧牲隱私的前提下安全運行。
