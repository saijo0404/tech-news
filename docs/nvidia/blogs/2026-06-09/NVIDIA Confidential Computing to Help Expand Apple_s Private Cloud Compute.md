# NVIDIA Confidential Computing to Help Expand Apple's Private Cloud Compute

- **來源**: NVIDIA Blogs
- **發布日期**: 2026-06-09
- **原文連結**: https://blogs.nvidia.com/blog/nvidia-confidential-computing-apple-private-cloud-compute/

## 核心主題
NVIDIA 宣布其 Confidential Computing 技術整合至 Apple Private Cloud Compute（PCC），支援 Apple 在 Google Cloud 基礎設施上使用 NVIDIA Blackwell GPU 進行保密推理，為 Apple Intelligence 提供硬體級別的資料隱私保護。

## 關鍵重點
- **NVIDIA Blackwell GPU 支援 Apple 基礎模型雲端推理**：Apple 在 WWDC 宣布 PCC 擴展至 Google Cloud，NVIDIA Blackwell GPU 將支援 Apple Foundation Models（由 Apple 與 Google 共同建構、基於 Gemini 模型技術的第三世代 Apple 基礎模型）的伺服器端推理，實現 Apple Intelligence 下一代功能。
- **Confidential Computing 硬體安全架構**：NVIDIA Confidential Computing 提供硬體級安全層，透過可信執行環境（TEE）隔離工作負載，使系統能在發送任何敏感資料前以密碼學方式驗證基礎設施未被篡改；涵蓋硬體信任根、加密通訊路徑、遠端證明，以及加速 AI 推理與訓練支援。
- **終端使用者隱私保障**：對最終使用者而言，即使系統建置者也无法查看使用者的資料、聊天或對話；隨著 AI 體驗結合裝置端與雲端處理，NVIDIA Confidential Computing 滿足高效能伺服器端推理與強隱私安全保證的雙重需求。

## 結論
這項由 NVIDIA、Apple 與 Google 三方協作的成果標誌著 AI 基礎設施隱私保護的重要里程碑，將 Confidential Computing 擴展至雲端環境，讓 Apple Intelligence 在利用強大 GPU 效能進行雲端推理的同時，確保使用者資料始終受到硬體層級的加密與隔離保護。
