# NVIDIA Vera Rubin Driving Performance Per Watt, Lowest Token Cost for Partners Worldwide

- **來源**: NVIDIA Blog
- **發布日期**: 2026-07-21
- **原文連結**: https://blogs.nvidia.com/blog/vera-rubin/

## 核心主題
NVIDIA Vera Rubin 平台以極端程式碼設計實現每瓦特最高效能與最低 token 成本，由 CoreWeave、Google Cloud、Microsoft Azure 和 Mistral 等全球夥伴率先部署。

## 關鍵重點
- **極端程式碼設計**：整合七顆晶片與五個托盤，Vera Rubin NVL72 採用客製 Olympus 核心，單線程效能提升 2 倍、記憶體延遲降低 40%
- **效能突破**：CoreWeave 測試顯示 Vera Rubin NVL72 比 Grace Blackwell NVL72 每兆瓦多 10 倍 token 吞吐量，解決 AI 工廠電力限制問題
- **專用網路架構**：第六代 NVLink 升級提供 2 倍吞吐量、3 倍更低延遲；Spectrum-X Ethernet 支援 102.4T 交換機系統，1.6 倍 RDMA 頻寬
- **綠色設計**：45°C 液冷入口溫度設計實現無冷卻器乾燥冷卻，每年每兆瓦節省數百萬加侖用水
- **歐洲開放模型生態**：Microsoft 與 Mistral 合作，提供符合歐洲法規的主權 AI 基礎設施，支持政府與監管產業使用

## 結論
Vera Rubin 平台透過極端程式碼設計與全球夥伴合作，重新定義 AI 基礎設施標準，同時滿足效能、成本、環保與區域控制的多重需求，為歐洲開放模型生態奠定基礎。
