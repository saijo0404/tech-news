# How Businesses Are Building Specialized AI They Can Trust

- **來源**: NVIDIA Blogs
- **發布日期**: 2026-06-23
- **原文連結**: https://blogs.nvidia.com/blog/nvidia-agent-toolkit-open-models-tools-skills-secure-runtime-ai-agents/

## 核心主題
NVIDIA Agent Toolkit 提供開放模組化基礎——整合可自訂模型、工具技能與安全執行環境，協助企業建構專注於自身工作流程的「專業 AI 同事」，讓 AI Agent 從試點實驗邁向可信任的實際生產應用。

## 關鍵重點
- **NVIDIA Agent Toolkit 三大核心組件**：（1）NVIDIA Nemotron 開放模型——提供推理基礎，讓團隊能自訂、評估並部署符合自身需求的 AI Agent；（2）NVIDIA Nemoclaw 藍圖與技能（Skills）——提供安全 Agent 行為模式，透過工具與技能將 Agent 連線至企業現有系統與領域專業知識，以更低成本達成準確結果；（3）NVIDIA OpenShell 安全執行環境——讓 Agent 能在工作系統內部安全運作。三者結合將前沿模型轉化為全功能數位同事，且支援與第三方 Agent 框架（如 Hermes Agents 與 OpenClaw）整合。
- **跨產業專業 Agent 應用現況**：生命科學領域，Agent 可呼叫蛋白質設計、虛擬篩選、基因組分析與生物標記發現等領域模型，BioNeMo Agent Toolkit 將以往需數月的研究縮短至數天；醫療健康領域，Agent 支援臨床文件、臨床決策支援與照護協調，數位雙生訓練的機器人 Agent 則擴展手術協助與醫院自動化；軟體、資安與工業營運方面，Agent 連線至團隊既有用具與數據，Cadence 與 Synopsys 正在建構全自動化晶片設計 Agent，CrowdStrike 的專業資安 Agent 以 98.5% 準確率分類警示，Palantir、SAP、ServiceNow、Siemens 與 Dassault Systèmes 則將 Agent 能力嵌入企業決策平台。
- **從通用存取到專業控制的產業轉向**：企業 AI 的第一波浪潮是「存取」——實驗新模型與進行試點；現在則轉向「專業」——建構能推理、使用工具並採取行動的 Agent 系統，服務最複雜的工作流程。最具有價值的 Agent 將是專業化的——它們結合模型、工具、技能、執行環境與基礎設施，讓企業能以自身工作流為核心自訂、控制並信任 AI。

## 結論
NVIDIA Agent Toolkit 以開放模組化架構回應企業對「可信任專業 AI」的需求——不再追求通用但不可控的 AI，而是讓企業擁有可自訂模型、連線現有工具、安全執行於內部系統的 AI 同事。從生命科學加速藥物發現到資安以 98.5% 準確率分類警示，專業 Agent 已證明其在實際工作場景中的價值，而 NVIDIA 提供的工具鏈讓更多企業能將 AI 從試點推向規模化部署。
