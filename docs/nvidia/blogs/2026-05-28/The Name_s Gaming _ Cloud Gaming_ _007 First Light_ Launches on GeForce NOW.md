# 007 First Light 登陸 GeForce NOW

- **來源**: NVIDIA Blog
- **發布日期**: 2026-05-27
- **原文連結**: https://blogs.nvidia.com/blog/geforce-now-thursday-007-first-light-launch/

## 核心主題
GeForce NOW 宣布 007 新作《007 First Light》上架雲端遊戲平台，玩家可使用任何裝置遊玩這款重新詮釋的龐德起源故事，同時本週還有八款新遊戲加入雲端遊戲庫。

## 關鍵重點
- **007 First Light 雲端上架**：玩家可在任何裝置上以高達 5K HDR 畫質體驗遊戲，Ultimate 會員可享 cinematic 品質串流，無需高階電腦。限時內購買一年 Ultimate 會員即含本作。
- **專屬獎勵與新遊戲**：Ultimate 會員可領取「Daring Elite Outfit」獨家造型；本週新增 Romestead、World of Tanks: HEAT、Starminer 等八款遊戲，同時推出《惡靈古堡：安魂曲》免費試玩版。
- **World of Tanks: HEAT 特色**：Wargaming 推出的英雄驅動坦克射擊遊戲，支援 5v5 與 10v10 快節奏對戰，每位 Agent 帶來獨特技能與載具陣容，現在也能在 GeForce NOW 上遊玩。

## 結論
GeForce NOW 持續擴展遊戲庫與雲端體驗，讓玩家輕鬆在各種裝置上享受高畫質遊戲，無論是龐德影迷還是坦克愛好者都能在本週找到值得嘗試的作品。
