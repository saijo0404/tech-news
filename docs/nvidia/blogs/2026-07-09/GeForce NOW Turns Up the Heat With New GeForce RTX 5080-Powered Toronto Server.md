# GeForce NOW Turns Up the Heat With New GeForce RTX 5080-Powered Toronto Server

- **來源**: blogs.nvidia.com/blog
- **發布日期**: 2026-07-09
- **原文連結**: https://blogs.nvidia.com/blog/geforce-now-thursday-toronto-expansion/

## 核心主題
NVIDIA GeForce NOW 雲端遊戲服務在加拿大多倫多新增 GeForce RTX 5080 伺服器，並推出多款新遊戲與功能更新，提升雲端遊戲體驗。

## 關鍵重點
- 多倫多新增 GeForce RTX 5080 伺服器，支援 4K 解析度、120 FPS、DLSS、光線追蹤與 Reflex 技術，讓玩家享受頂級雲端遊戲性能
- NTE: Neverness to Everness 推出「999 Nights」版本，新增永久遊戲模式、新角色與摩托車，並即將支援原生觸控控制
- 新增三款遊戲：Esports Manager 2026、Assassin's Creed Black Flag Resynced 與 Granblue Fantasy: Relink – Endless Ragnarok Demo

## 結論
GeForce NOW 持續擴展雲端遊戲服務，透過硬體升級與新遊戲內容，讓更多玩家無須下載或額外硬體即可享受高品質遊戲體驗。
