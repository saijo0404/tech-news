# NVIDIA DSX OS 提供開源模組化軟體，實現大規模 AI 工廠營運

- **來源**: NVIDIA Developer Blog
- **發布日期**: 2026-05-31
- **原文連結**: https://developer.nvidia.com/blog/nvidia-dsx-os-delivers-open-modular-software-for-operating-ai-factories-at-scale/

## 核心主題
NVIDIA 發布 DSX OS 開源模組化軟體平台，將 AI 工廠的能源、晶片、基礎設施、模型與應用程式五層堆疊整合於統一的共同設計架構，以加速部署、提升效能與強化營運韌性。

## 關鍵重點
- **三大營運成果驅動 AI 工廠經濟**：DSX OS 讓生態系夥伴直接採用 NVIDIA 自身營運的基礎設施軟體而非重新開發，將部署時間縮短數月；在固定電力預算下最多增加 40% GPU 數量而不影響推論效能；並將叢集營運從被動告警轉變為自動化修復。
- **五大核心能力覆蓋 Gigawatt 級工廠**：DSX Exchange 以 MQTT 橋接 IT/OT 通訊讓代理式 AI 可發現完整工廠操作面；DSX MaxLPS 與 Flex 將電力視為可程式化資源並連接電網服務；Infra Controller 與 AICR 提供 API 驅動的裸機生命週期管理與版本鎖定配方；NVSentinel 與 Fleet Intelligence 實現 Kubernetes 原生 GPU 故障檢測與全球可見性；KAI Scheduler、Dynamo、Grove 與 NVCF 提供智能排程與分散式推論 API。
- **生態系夥伴全面採用**：CoreWeave、Firmus、Lambda、Emerald AI、IREN、Red Hat、Supermicro、Lambda 等夥伴已部署或整合 DSX OS 各元件，涵蓋從電力最佳化、多租戶生命週期到健康監控與智能排程的完整營運鏈。

## 結論
DSX OS 以開源、模組化設計讓 AI 工廠營運從人工排程邁向代理式自動化，夥伴可根據自身需求逐步採用各元件，加速部署並提升營運效率，為大規模代理式 AI 工廠提供堅實的操作系統基礎。
