# 使用 AI 代理技能與 NVIDIA Nemotron Speech 加速臨床語音辨識模型評估

- **來源**: NVIDIA Developer Blog
- **發布日期**: 2026-06-09
- **原文連結**: https://developer.nvidia.com/blog/evaluate-clinical-asr-models-faster-with-agent-skills-and-nvidia-nemotron-speech/

## 核心主題
本文介紹一套結合 NVIDIA AI 代理技能、NeMo Data Designer 與 Nemotron Speech 的自動化工作流，透過產生音標感知的合成語音資料，加速臨床語音辨識（ASR）模型的評估與改進循環。

## 關鍵重點
- **以臨床資料檔案為核心的質改善迴圈**：開發者透過簡單的對話設定臨床專科檔案（如骨科、心臟科、腫瘤科等），代理技能自動建立包含藥物、手術、解剖學與疾病術語的基準測試，並透過評估結果決定下一步是擴展詞彙、改善發音或微調模型，形成「建立 → 評估 → 改進 → 重新評估」的持續循環。
- **音標注入與人工發音審查機制**：使用 SSML 標記將 IPA（國際音標）注入合成語音，對於詞典中找不到的術語，代理會提出候選發音並等待人工審查確認後再生成音訊，確保合成資料的發音正確性，避免錯誤發音污染訓練資料。
- **實體層級評估指標驅動決策**：除了傳統的詞錯誤率（WER）和字錯誤率（CER），評估技能提供關鍵詞錯誤率（KER）等指標，讓開發者能精準定位特定類別（如藥物名稱）的弱點，並據此採取有針對性的改進措施，而非盲目進行模型微調。

## 結論
這個技能驅動的臨床 ASR 質改善迴圈將原本耗時的語音模型評估流程自動化，透過合成語音、人工審查閘道和實體層級評估指標，讓開發者能快速定位並解決臨床術語辨識問題，為醫療語音 AI 的開發提供了一套可重複、可擴展的實踐框架。
