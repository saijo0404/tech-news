# Start Customizing NVIDIA Nemotron 3 Nano with Prime Intellect Lab in Minutes

- **來源**: NVIDIA Technical Blog
- **發布日期**: 2026-07-23
- **原文連結**: https://developer.nvidia.com/blog/start-customizing-nvidia-nemotron-3-nano-with-prime-intellect-lab-in-minutes/

## 核心主題
這篇文章介紹如何使用 Prime Intellect Lab 在短短數分鐘內，透過強化學習（RL）自訂 NVIDIA Nemotron 3 Nano 模型，並產生可下載的 LoRA 適配器。

## 關鍵重點
- 使用 Prime Intellect Lab 平台，可快速設定強化學習環境（約5分鐘），無需管理本地 GPU 集群。
- 以 Python Math 環境為範例，展示從基線評估、訓練到重新評估的完整流程。
- 訓練後準確率從 21.9% 大幅提升至 90.6%，證明自訂模型效果顯著。
- NVIDIA Nemotron 3 系列模型（Nano/Super/Ultra）均支援此工作流程，且開源權重、數據與訓練配方確保可重現性。

## 結論
透過 Prime Intellect Lab 等雲端平台，開發者如今可以極易地自訂開源大模型，大幅降低技術門檻並加速特定任務的模型優化。
