# NVIDIA-Verified Agent Skills Provide Capability Governance for AI Agents

- **來源**: developer  
- **發布日期**: 2026-05-19  
- **原文連結**: https://developer.nvidia.com/blog/nvidia-verified-agent-skills-provide-capability-governance-for-ai-agents/  

## 核心主題  
NVIDIA 推出驗證代理技能，透過透明化、風險掃描與加密簽章，提升 AI 代理能力的可信度與安全性。  

## 關鍵重點  
- 驗證技能包含每日更新目錄、自動化與人工審核、風險掃描及加密簽章，確保技能來源與完整性。  
- 技能卡集中信任元資料，協助開發者評估相容性、風險與驗證狀態，降低部署風險。  
- 透過 SkillSpector 工具掃描軟體風險與代理特有風險，並結合加密簽章保障技能 authenticity。  

## 結論  
NVIDIA 驗證技能強化 AI 代理治理，使開發者能更安全、透明地擴展與部署 AI 能力。
