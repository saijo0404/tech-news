# NVIDIA Ising Decoding Cuts Color Code Logical Error Rates by Over 300x

- **來源**: NVIDIA Technical Blog
- **發布日期**: 2026-07-13
- **原文連結**: https://developer.nvidia.com/blog/nvidia-ising-decoding-cuts-color-code-logical-error-rates-by-over-300x/

## 核心主題
NVIDIA 推出 Ising Decoder ColorCode 1 Fast，將 Color Code 量子錯誤糾正解碼器的邏輯錯誤率降低超過 347.7 倍，並提升 7.3 倍運行速度，使 Color Code 重新成為實用選擇。

## 關鍵重點
- Ising Decoder ColorCode 1 Fast 相比現有最佳解碼器 Chromobius，邏輯錯誤率 (LER) 降低超過 347.7 倍，運行速度快 7.3 倍
- 採用 3D 卷積神經網路 (CNN) 預解碼器，支援三角形 Color Code，可針對特定 QPU 架構和噪音特徵進行客製化
- NVIDIA 提供完整的開源資源，包括權重、訓練食譜、合成資料生成工具 (cuQuantum/cuStabilizer) 和完整訓練管道

## 結論
NVIDIA 的 Ising 解碼器技術讓 Color Code 量子錯誤糾正重新成為可實用的選擇，為建構真正有用的量子電腦打開了新途徑。
