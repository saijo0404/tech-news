# DynoSim：模擬帕累托最優前沿

- **來源**: NVIDIA Developer Blog
- **發布日期**: 2026-05-29
- **原文連結**: https://developer.nvidia.com/blog/dynosim-simulating-the-pareto-frontier/

## 核心主題
DynoSim 是 NVIDIA Dynamo 推理服務平台的離散事件模擬器，能忠實模擬整個服務堆疊，讓開發者在部署前快速評估數千種配置組合，找到效能與成本的最佳平衡點。

## 關鍵重點
- **模擬即驗證循環**：DynoSim 將工作負載重播、引擎排程、路由、Planner 決策與 KV 快取行為整合在同一虛擬時間軸上，單一-thread 模擬 23,608 個請求僅需 2.41 秒，比真實時間快約 1,500 倍。
- **排程器與多引擎模擬**：透過 AIConfigurator 提供精準的引擎側效能估算，並建模不同後端（vLLM、SGLang）的排程行為；多引擎模擬則捕捉路由、分佈式快取等跨工作器的互動決策。
- **自動化最佳化與研究發現**：DynoSim 可系統性掃描部署參數，甚至以自動化研究方式探索新的路由成本函數與快取策略；實驗顯示動態部署搭配 SLA 目標的 Planner 能以更少 GPU 資源達成更低的 TTFT 與 ITL，擴展間隔以 5-10 秒為最佳。

## 結論
DynoSim 讓模擬成為部署調整的內循環、真實叢集成為驗證的外循環，大幅降低探索服務算法的成本，並能持續追蹤生產環境的負載漂移，自動推薦最佳部署配置。
