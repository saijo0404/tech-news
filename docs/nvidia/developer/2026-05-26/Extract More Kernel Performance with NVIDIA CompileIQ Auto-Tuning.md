# 利用 NVIDIA CompileIQ 自動調優提取更多核函式效能

- **來源**: NVIDIA Developer Blog
- **發布日期**: 2026-05-25
- **原文連結**: https://developer.nvidia.com/blog/extract-more-kernel-performance-with-nvidia-compileiq-auto-tuning/

## 核心主題
NVIDIA 推出 CompileIQ，一個 AI 驅動的編譯器自動調優框架，利用進化與遺傳演算法針對特定工作負載自動優化 NVIDIA GPU 編譯器內部參數，突破傳統優化手段的極限。

## 關鍵重點
- **解決的痛點**：當開發團隊已耗盡所有優化手段（batch size、量化、Flash Attention、核函式融合等）後，編譯器本身的內部參數仍有一個巨大的優化空間未被利用。
- **90% 的計算占比**：在 LLM 推理中，Attention 核函式、GEMM 以及 QKV 投影等核函式佔總運算量超過 90%，這些小段代碼對整體性能影響極大。
- **CompileIQ 原理**：通過進化與遺傳演算法在編譯器內部參數空間（寄存器分配策略、指令排程策略、循環轉換等）中搜尋，生成針對特定核函式優化的進階控制文件（ACF）。
- **使用方法簡單**：只需安裝 `compileiq` Python 套件，定義目標函數（編譯並評估核函式效能），讓 CompileIQ 自動搜尋最佳編譯器配置。
- **性能提升明顯**：已在生產環境中驗證，對 Triton 與 Helion 核函式帶來最多達 15% 的性能提升。
- **多目標優化與 IP 保護**：支援同時優化多個目標（如運算時程、編譯時程、功耗），且用戶負載永遠保留在本地環境，僅輸出安全的 ACF 文件，可安全提交至版本控制系統。

## 結論
CompileIQ 讓開發者不再被限制在傳統優化手段的範圍內，將編譯器本身變成另一個可調整的參數。對於已耗盡所有優化手段的團隊，這是新的突破點；對於未達極致的團隊，則是進一步榨取性能的利器。
